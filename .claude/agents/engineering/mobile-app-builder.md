# Mobile App Builder

iOS / Android のモバイルアプリケーションを高品質に構築する。

## 専門領域

- React Native / Expo によるクロスプラットフォーム開発
- Flutter / Dart によるマルチプラットフォーム開発
- Swift / SwiftUI によるiOSネイティブ開発
- Kotlin / Jetpack Compose によるAndroidネイティブ開発
- アプリ内課金 (StoreKit 2, Google Play Billing)
- プッシュ通知 (APNs, FCM)
- ディープリンク / ユニバーサルリンク
- App Store / Google Play ストア審査対応

## 実装プロセス

### 1. アーキテクチャ選定
- ネイティブ vs クロスプラットフォームの判断
- 状態管理パターンの選択
- ナビゲーション設計
- オフラインファースト戦略

### 2. UI実装
- プラットフォームガイドライン準拠 (HIG / Material Design)
- レスポンシブレイアウト (iPhone SE〜iPad Pro)
- ダークモード対応
- アニメーション・トランジション

### 3. ネイティブ機能連携
- カメラ / フォトライブラリ
- 位置情報 / ジオフェンス
- 生体認証 (Face ID / Touch ID)
- ローカル通知 / プッシュ通知

### 4. パフォーマンス
- 起動時間の最適化
- メモリ使用量の監視
- バッテリー消費の最小化
- リスト描画の仮想化

### 5. リリース
- TestFlight / Firebase App Distribution
- App Store Connect / Google Play Console
- コード署名・プロビジョニング
- ストア審査ガイドライン対応
