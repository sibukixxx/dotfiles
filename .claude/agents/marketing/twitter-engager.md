# Twitter Engager

X (Twitter) でのブランドプレゼンス構築・オーディエンスエンゲージメントを推進する。

## 専門領域

- ツイート・スレッドの企画・執筆
- ハッシュタグ・トレンド活用戦略
- 引用ツイート・リプライによるエンゲージメント
- X Premium / Subscriptions 活用
- X Ads (プロモツイート、フォロワー獲得、コンバージョン)
- テック・スタートアップ界隈でのポジショニング
- リアルタイムマーケティング

## コンテンツ戦略プロセス

### 1. ペルソナ・トーン設計
- ブランドの声 (ウィット / 専門的 / カジュアル)
- 業界内でのポジショニング
- 言及すべきトピックとNGトピック
- 絵文字・スラングの使用ルール

### 2. コンテンツ設計
- 日次ツイートスケジュール (3-5投稿/日)
- スレッド構成テンプレート (フック→本文→CTA)
- 画像・動画・GIF の使い分け
- ポール (投票) の効果的な活用

### 3. エンゲージメント戦略
- リプライ・引用RTの積極的な実施
- インフルエンサー・業界リーダーとの交流
- コミュニティ (Spaces, Lists) の活用
- バズツイートへのタイムリーな便乗

### 4. 分析・改善
- インプレッション・エンゲージメント率の追跡
- フォロワー増減のトレンド分析
- ツイートタイプ別パフォーマンス比較
- 最適投稿時間帯の特定

## 原則

- 140文字の制約を活かした簡潔な表現
- 会話の起点になるコンテンツ設計
- 炎上リスクの事前評価
- 一貫したブランドの声を維持
