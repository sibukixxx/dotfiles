# 契約書レビューチェックリスト

## 契約レビュー（日本語）

### 最優先チェック項目

| カテゴリ | チェック項目 | リスク |
|---------|------------|--------|
| 損害賠償 | 上限が設定されているか | 無制限は危険 |
| 損害賠償 | 間接損害・逸失利益の扱い | 除外条項の有無 |
| 契約解除 | 解除条件が一方的でないか | 相手方のみ解除権あるケース |
| 契約解除 | 解除時の精算方法 | 既払い金の返還規定 |
| 知財 | 権利帰属が明確か | 曖昧だと紛争の種 |
| 知財 | 著作者人格権の扱い | 不行使特約の有無 |
| 秘密保持 | 残存期間は適切か | 永久は負担大 |
| 競業避止 | 範囲・期間が合理的か | 過度な制限は無効の可能性 |
| 自動更新 | 更新条件・解約告知期間 | 気づかず更新されるリスク |
| 準拠法 | 外国法が指定されていないか | 日本法以外は専門家確認 |

### 一般チェック項目

- [ ] 当事者の表記が正確か（商号・住所・代表者）
- [ ] 契約期間・有効期限が明確か
- [ ] 支払条件（金額・期限・方法）が具体的か
- [ ] 不可抗力条項があるか
- [ ] 反社会的勢力排除条項があるか
- [ ] 個人情報の取扱いが規定されているか
- [ ] 通知方法（書面・メール等）が定められているか
- [ ] 協議条項があるか
- [ ] 管轄裁判所が自社に不利でないか

## 英文契約レビュー

### 日本企業が注意すべき「落とし穴」

#### 1. Limitation of Liability（責任制限）

チェックポイント:
- **Cap（上限）**: 契約金額の何倍か。"aggregate liability" の定義
- **Exclusions**: consequential, indirect, incidental, special damages の除外範囲
- **Carve-outs**: 秘密保持違反・知財侵害は上限の例外か
- **Mutual or one-sided**: 双方向か片方向か

危険パターン:
- 相手方の責任上限が極端に低い（例: 直近12ヶ月の支払額）
- 自社の責任は無制限で相手方のみ制限あり

#### 2. Indemnification（補償）

チェックポイント:
- **Scope**: 何に対する補償か（第三者クレーム、知財侵害等）
- **Defense obligation**: 防御義務の有無
- **Control**: 訴訟の主導権は誰が持つか
- **Settlement**: 和解の権限

危険パターン:
- "indemnify, defend, and hold harmless" のフルセット要求
- 自社のみが補償義務を負う片務的条項

#### 3. Governing Law & Dispute Resolution（準拠法・紛争解決）

チェックポイント:
- 準拠法が相手方の国の法律になっていないか
- 仲裁 vs 裁判所の選択
- 仲裁地・仲裁機関の指定

推奨:
- 日本法を準拠法にする交渉
- 困難な場合はシンガポール国際仲裁センター（SIAC）等の中立地

#### 4. その他の注意点

- **Entire Agreement（完全合意条項）**: 口頭の約束が無効になる
- **Assignment（譲渡）**: 同意なしの契約譲渡が可能になっていないか
- **Force Majeure（不可抗力）**: パンデミック・サイバー攻撃が含まれるか
- **Termination for Convenience（任意解除）**: 相手方のみの随時解除権は危険
- **Non-solicitation（勧誘禁止）**: 従業員の引き抜き禁止の範囲
