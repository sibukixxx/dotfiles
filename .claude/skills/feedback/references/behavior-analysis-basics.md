# 行動分析学の基礎

## 行動分析学とは

行動分析学（Behavior Analysis）は、B.F.スキナーによって体系化された、行動の原理を科学的に研究する学問。人間や動物の行動がどのように環境によって形成・維持されるかを分析する。

## 基本原理

### 1. オペラント条件づけ

行動は、その結果（強化・罰）によって制御される。

```
行動 → 結果（強化）→ 行動頻度増加
行動 → 結果（罰）  → 行動頻度減少
```

### 2. レスポンデント条件づけ

刺激と反応の連合学習（パブロフの犬）。

```
無条件刺激 → 無条件反応
条件刺激（繰り返し対呈示）→ 条件反応
```

### 3. 三項随伴性（ABC分析）

```
Antecedent (先行条件)
    ↓
Behavior (行動)
    ↓
Consequence (結果)
```

## 強化と罰

### 正の強化（Positive Reinforcement）

行動後に**好ましい刺激を加える**ことで行動を増やす。

| 例 | 行動 | 結果 |
|---|---|---|
| アプリ | タスク完了 | バッジ獲得 |
| EC | 購入 | ポイント付与 |
| SNS | 投稿 | いいね獲得 |

### 負の強化（Negative Reinforcement）

行動後に**嫌悪刺激を除去する**ことで行動を増やす。

| 例 | 行動 | 結果 |
|---|---|---|
| アプリ | 設定完了 | 警告メッセージ消失 |
| EC | 会員登録 | 広告非表示 |
| 通知 | 確認 | バッジ消失 |

### 正の罰（Positive Punishment）

行動後に**嫌悪刺激を加える**ことで行動を減らす。

| 例 | 行動 | 結果 |
|---|---|---|
| ゲーム | ミス | ライフ減少表示 |
| フォーム | 不正入力 | エラーメッセージ |

### 負の罰（Negative Punishment）

行動後に**好ましい刺激を除去する**ことで行動を減らす。

| 例 | 行動 | 結果 |
|---|---|---|
| サービス | 規約違反 | アカウント停止 |
| ゲーム | 離脱 | ストリーク消失 |

## 企画設計への応用

### 効果的な強化の条件

1. **即時性**: 行動直後に結果を提示
2. **随伴性**: 行動と結果の明確な対応関係
3. **価値**: ユーザーにとって意味のある報酬
4. **適切な量**: 過剰でも過少でもない

### 避けるべきパターン

| パターン | 問題点 |
|----------|--------|
| 遅延した報酬 | 行動との結びつきが弱い |
| 確率的報酬（のみ） | 行動の消去が起きやすい |
| 内発的動機の阻害 | 過剰な外的報酬は内発的動機を低下させる |
| 罰への依存 | 監視がないと効果がなくなる |

## 行動の機能分析

行動には目的（機能）がある。問題行動を分析する際の4つの機能：

| 機能 | 説明 | 例 |
|------|------|-----|
| 注目獲得 | 他者からの注目を得る | SNSでの過激な投稿 |
| 逃避・回避 | 嫌な状況から逃れる | タスクのスキップ |
| 物・活動へのアクセス | 欲しいものを得る | 広告スキップのための課金 |
| 感覚刺激 | 感覚的な快を得る | スクロールし続ける |
