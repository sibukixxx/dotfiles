# 競合・伸び投稿の観察（毎朝10分）

目的：「すごいな」で流し見せず、**なぜ売れているのか**を毎日1行ずつ言語化して市場感覚を作る。

## 手順

### 1. 今日見る場所を1つ決めてもらう
毎日全部は見ない。ローテーションで1〜2箇所に絞らせる：

- Xで伸びている人 / Instagramで売れている人 / TikTokで集客している人
- YouTubeで案件を取っている人 / note・Brain・Tipsで売れている人
- ココナラ・クラウドワークスで受注している人 / LP・メルマガ・LINE誘導している人

ユーザーの参入ジャンルが分かっていれば、そのジャンルで具体的に「誰を・どんな検索で見るか」まで提案する。

### 2. 競合の投稿を10個ざっと見て、3個選んでもらう
選ぶ基準（保存すべき投稿）：

- いいね・ブックマーク・コメントが多い／引用されている／賛否が起きている
- 商品販売やプロフィール遷移につながっていそう
- 自分でも真似できそう

### 3. 貼ってもらった投稿を分解する
スクショ・コピペ・要約のどれでも受け付ける。1投稿につきこのテンプレで**Claudeが下書きを作り、ユーザーに1行直してもらう**：

```
投稿テーマ：
誰向け：
刺さっている悩み：
使っている言葉（原文の言い回し）：
伸びた理由（1行）：
自分ならどう応用するか（1行）：
```

分解の観点（テンプレを埋めるときに意識する）：

- どんな悩みを扱い、誰に向けて、どんな不安と未来を見せているか
- どんな実績を見せ、無料でどこまで出し、有料へどう誘導しているか
- コメント欄で何を言われているか（共感／質問／「まさにこれ」／「何から始めれば」系は商品テーマの原石として必ず拾う）

### 4. 「刺さる言葉」を収集する
投稿・コメントに出てきた反応されている言葉（例：未経験から／月5万円／時短／自動化／丸投げ／再現性／ロードマップ）を見つけたらリストに追加する。ポイントは**自分が言いたい言葉ではなく、相手が反応している言葉**を貯めること。

### 5. ログを出力する
3件分そろったら（少なくても可）、`/mnt/user-data/outputs/投稿観察_YYYY-MM-DD.md` に保存して `present_files` で渡す。日付は `user_time_v0` で確認する。

## 注意

- 分析が長文にならないこと。1投稿につきテンプレ6行で完結させる。
- 投稿が1個しか出てこない日でもOKとし、継続を優先する。
- 週に一度（月曜など）は「今週見返すと、伸びていたテーマは〇〇系」という横断コメントを添える。
