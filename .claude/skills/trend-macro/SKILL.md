# Trend Macro - マクロトレンド分析

「自分が知らないことを知る」ためのマクロトレンド分析。
普段触れない領域の動向を6カテゴリで俯瞰し、視野を広げる。

## 設計思想

### なぜこのスキルか

- **フィルターバブル脱出**: 自分の興味外の情報に触れる
- **大局観の獲得**: 個別ニュースでなく構造的な変化を捉える
- **意思決定の質向上**: マクロ環境を理解して判断する

### 情報ソース

| ソース | 用途 | 特徴 |
|--------|------|------|
| **WebSearch** | 最新トレンド・深掘り | リアルタイム性 |
| **NHK RSS** | 国内ニュース基盤 | 信頼性・網羅性 |

※ Inoreaderは使用しない（個人購読フィードは別スキルで）

## モード選択

```
/trend-macro           → 週間モード（デフォルト）
/trend-macro --weekly  → 週間モード
/trend-macro --monthly → 月間モード
```

| モード | 対象期間 | 粒度 | 用途 |
|--------|----------|------|------|
| **weekly** | 直近1週間 | 個別トピック | 週次キャッチアップ |
| **monthly** | 直近1ヶ月 | 構造的変化 | 月次振り返り |

## 6カテゴリ

```
┌─────────────────────────────────────────────────────┐
│                  マクロトレンド                      │
├──────────┬──────────┬──────────┬──────────┬─────────┤
│  政治    │  経済    │  社会    │  科学    │  環境   │
│ Politics │ Economy  │ Society  │ Science  │ Climate │
├──────────┴──────────┴──────────┴──────────┴─────────┤
│                      産業 Industry                   │
└─────────────────────────────────────────────────────┘
```

### カテゴリ定義

| カテゴリ | スコープ | キーワード例 |
|----------|----------|-------------|
| **政治** | 国際関係、政策、選挙、規制 | 外交、安全保障、法改正 |
| **経済** | 金融、市場、雇用、貿易 | 金利、為替、GDP、インフレ |
| **社会** | 人口動態、文化、教育、医療 | 少子化、働き方、格差 |
| **科学** | 研究、技術革新、宇宙、医学 | AI、量子、創薬、論文 |
| **環境** | 気候変動、エネルギー、災害 | 脱炭素、再エネ、COP |
| **産業** | 業界動向、企業、スタートアップ | M&A、IPO、規制、DX |

## 実行プロセス

### Step 1: NHK RSSから国内ニュース取得

**使用するRSSフィード:**

```
主要ニュース: https://www.nhk.or.jp/rss/news/cat0.xml
社会:         https://www.nhk.or.jp/rss/news/cat1.xml
科学・文化:   https://www.nhk.or.jp/rss/news/cat3.xml
政治:         https://www.nhk.or.jp/rss/news/cat4.xml
経済:         https://www.nhk.or.jp/rss/news/cat5.xml
国際:         https://www.nhk.or.jp/rss/news/cat6.xml
```

**取得方法:**
```
WebFetch で各RSSを取得し、直近の記事タイトル・概要を抽出
```

### Step 2: WebSearchで補完・深掘り

**カテゴリ別検索クエリ例:**

| カテゴリ | 検索クエリ |
|----------|-----------|
| 政治 | `"国際情勢" OR "外交" 2024年` |
| 経済 | `"経済動向" OR "金融市場" 最新` |
| 社会 | `"社会問題" OR "人口動態" トレンド` |
| 科学 | `"科学技術" OR "研究開発" 最新` |
| 環境 | `"気候変動" OR "環境政策" 2024年` |
| 産業 | `"業界動向" OR "企業戦略" 最新` |

### Step 3: トレンド抽出・構造化

**分析の観点:**

1. **変化の兆し**: 新しく出てきた話題は何か
2. **加速する動き**: 勢いを増している流れは何か
3. **転換点**: 潮目が変わりそうな事象は何か
4. **相互連関**: カテゴリ間でつながる話題は何か

## 使用ツール

### WebFetch（RSS取得）

```javascript
// NHK RSSフィードを取得
WebFetch({
  url: "https://www.nhk.or.jp/rss/news/cat0.xml",
  prompt: "記事のタイトル、リンク、公開日を抽出してください"
})
```

### WebSearch（深掘り・補完）

```javascript
// カテゴリ別に検索
WebSearch({
  query: "国際情勢 2024年 重要ニュース"
})
```
