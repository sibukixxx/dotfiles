# 6カテゴリ詳細定義

マクロトレンド分析で使用する6カテゴリの詳細定義と分析観点。

## 1. 政治（Politics）

### スコープ

| 領域 | 含むもの |
|------|----------|
| 国際関係 | 外交、同盟、対立、条約、サミット |
| 安全保障 | 防衛、軍事、テロ、サイバー |
| 国内政治 | 選挙、政党、内閣、議会 |
| 規制・法制度 | 法改正、規制強化/緩和、司法判断 |
| 地政学 | 地域覇権、資源争奪、通商摩擦 |

### 分析観点

- **パワーバランスの変化**: 国際的な力関係に変化はあるか
- **政策転換のシグナル**: 大きな方針転換の兆しはあるか
- **規制環境の変化**: ビジネスに影響する規制動向は
- **リスク要因**: 地政学リスク、政情不安の兆候

### 注目すべき指標

- 主要国の支持率・政権安定度
- 国際条約・協定の動向
- 制裁・関税の発動状況
- 選挙スケジュール

## 2. 経済（Economy）

### スコープ

| 領域 | 含むもの |
|------|----------|
| マクロ経済 | GDP、インフレ、雇用、消費 |
| 金融政策 | 金利、量的緩和、中央銀行動向 |
| 市場 | 株式、債券、為替、コモディティ |
| 貿易 | 輸出入、サプライチェーン、関税 |
| 財政 | 政府支出、税制、債務 |

### 分析観点

- **景気サイクルの位置**: 拡大期/後退期/回復期のどこか
- **金融環境の変化**: 金利・流動性の方向性
- **インフレ動向**: 物価上昇圧力の強さ
- **成長ドライバー**: 何が経済を牽引しているか

### 注目すべき指標

| 指標 | 意味 |
|------|------|
| GDP成長率 | 経済全体の成長 |
| CPI/コアCPI | インフレ率 |
| 失業率 | 雇用環境 |
| 政策金利 | 金融政策スタンス |
| PMI | 景況感 |
| 長短金利差 | 景気後退シグナル |

## 3. 社会（Society）

### スコープ

| 領域 | 含むもの |
|------|----------|
| 人口動態 | 少子高齢化、移民、都市化 |
| 労働 | 働き方、雇用形態、賃金 |
| 教育 | 学校教育、リスキリング、格差 |
| 医療・健康 | 公衆衛生、医療制度、メンタルヘルス |
| 文化・価値観 | 世代間ギャップ、多様性、消費行動 |
| 格差・貧困 | 所得格差、機会格差、社会的分断 |

### 分析観点

- **人口構造の変化**: 人口動態が社会にどう影響するか
- **価値観の変化**: 新しい世代の価値観は何か
- **社会的課題**: 顕在化している/しそうな問題は
- **行動変容**: 人々の行動パターンの変化

### 注目すべきテーマ

- 少子化対策と効果
- 働き方改革の実態
- Z世代・α世代の価値観
- 地方と都市の格差
- メンタルヘルス問題

## 4. 科学（Science）

### スコープ

| 領域 | 含むもの |
|------|----------|
| AI・情報技術 | 機械学習、生成AI、量子コンピュータ |
| バイオ・医療 | 創薬、遺伝子、再生医療 |
| 宇宙 | 探査、衛星、民間宇宙開発 |
| 材料・エネルギー | 新素材、電池、核融合 |
| 基礎研究 | ノーベル賞級の発見、新理論 |

### 分析観点

- **ブレークスルー**: 画期的な発見・発明はあるか
- **実用化の進展**: 研究から応用へのステージ
- **研究トレンド**: 資金・人材が集まっている分野
- **倫理的議論**: 技術の社会的影響への懸念

### 注目すべき動向

- 生成AIの進化と応用
- 量子コンピュータの実用化
- 創薬・医療技術の進歩
- 宇宙開発の民主化
- 研究不正・再現性問題

## 5. 環境（Climate/Environment）

### スコープ

| 領域 | 含むもの |
|------|----------|
| 気候変動 | 温暖化、異常気象、海面上昇 |
| エネルギー転換 | 再エネ、脱炭素、原子力 |
| 環境政策 | カーボンプライシング、規制 |
| 自然災害 | 地震、台風、洪水 |
| 生態系 | 生物多様性、森林、海洋 |

### 分析観点

- **気候変動の進行**: 観測データは何を示しているか
- **政策の進捗**: 各国のコミットメントと実態
- **技術の進歩**: 脱炭素技術の成熟度
- **経済との両立**: グリーン成長の可能性

### 注目すべき指標

| 指標 | 意味 |
|------|------|
| 世界平均気温 | 温暖化の進行 |
| CO2濃度 | 大気中の温室効果ガス |
| 再エネ比率 | エネルギー転換の進捗 |
| EVシェア | 運輸部門の脱炭素 |
| ESG投資額 | 資金の流れ |

## 6. 産業（Industry）

### スコープ

| 領域 | 含むもの |
|------|----------|
| テクノロジー | GAFAM、スタートアップ、VC |
| 製造業 | 自動車、半導体、素材 |
| 金融 | 銀行、保険、フィンテック |
| 小売・消費財 | EC、D2C、消費トレンド |
| エネルギー | 石油・ガス、電力、再エネ事業 |
| ヘルスケア | 製薬、医療機器、ヘルステック |

### 分析観点

- **業界再編**: M&A、統合、新規参入の動き
- **破壊的イノベーション**: 既存業界を変える動き
- **規制と対応**: 新規制への業界対応
- **サプライチェーン**: 供給網の変化・リスク

### 注目すべき動向

- ビッグテックの動向と規制
- 半導体サプライチェーン
- EV・自動運転の競争
- 生成AIの産業応用
- スタートアップ資金調達環境
