## 情報収集のベストプラクティス

### 1. 複数ソースのクロスチェック

```
単一ソース → バイアスの可能性あり
複数ソース → より客観的な把握
```

### 2. 一次ソースへの遡り

```
ニュース記事 → 引用されている一次ソース（統計、論文、公式発表）を確認
```

### 3. 時系列での変化を追う

```
単発のニュース → 点の情報
時系列で追う → 線（トレンド）として把握
```

### 4. 「ない」ことにも注目

```
報道されていないこと → なぜ報道されないか
減少したトピック → なぜ減少したか
```

## 情報の信頼性評価

### チェックポイント

| 観点 | 確認事項 |
|------|----------|
| ソース | 一次情報か二次情報か |
| 発信者 | 専門家か、利害関係者か |
| 日付 | 最新か、古い情報か |
| 裏付け | 他ソースで確認できるか |
| バイアス | 特定の立場に偏っていないか |

### 信頼度レベル

| レベル | 説明 | 例 |
|--------|------|-----|
| 高 | 公式発表、査読論文 | 政府統計、Nature |
| 中 | 大手メディア、専門誌 | NHK、日経 |
| 低 | 個人ブログ、SNS | 要クロスチェック |
