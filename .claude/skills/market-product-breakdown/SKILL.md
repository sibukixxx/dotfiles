# 売れている商品の分解（毎朝10分）

目的：投稿より商品を見る。売れている商品は「スキル」ではなく**「悩みの解決策」**として売られている。その型を毎日1個ずつ盗む。

## 手順

### 1. 今日分解する商品を1個決めてもらう
候補：note / Brain / Tips / Udemy / Kindle / ココナラ商品 / LP商品 / オンライン講座 / コンサル / サブスク / テンプレ販売 / 代行サービス。

商品ページ・LP・販売文・レビューのコピペ（またはスクショ・要約）を貼ってもらう。URLだけ貼られた場合は `web_fetch` で取得を試み、取得できなければ内容を貼ってもらう。

### 2. 商品を10項目で分解する

```
何を売っているか：
いくらか：
誰に売っているか：
商品名の付け方：
解決している悩み：
見せている実績：
特典：
保証：
レビューで褒められている点：
購入者が価値を感じている点：
```

分からない項目は空欄でよい。埋まらない項目こそ「次に自分が調べる観点」として指摘する。

### 3. 「スキル→解決策」の言い換えを確認する
その商品がどう言い換えているかを1行で指摘する。例：

- ライティングスキル → 売れるLPを作る
- デザインスキル → 申し込み率が上がるバナー
- AIスキル → 投稿作成を3時間から30分に

### 4. 低評価レビューがあれば必ず読む
レビューが貼られたら、高評価より低評価を重視して不満を抜き出す（難しすぎた／初心者向けでない／具体例が少ない／サポートが弱い／導入が面倒 など）。そして**「次に作るべき商品はこの不満の中にある」**という視点で、改善版の商品案を1つ提案する。

### 5. 価格の階段を描く
同ジャンルの価格帯をユーザーと一緒に整理する。例（SNS運用）：

```
投稿作成だけ：月3〜10万
企画込み：月10〜30万
分析込み：月20〜50万
広告運用込み：月30万〜
戦略設計込み：さらに上
```

「何が加わると単価が上がるか」を1行で言語化する。月額化・成果報酬・初期費用＋月額・パッケージ化の余地も一言添える。

### 6. 導線が見える場合は分解する
プロフィール→無料プレゼント→LINE/メルマガ→教育→個別相談→販売、のような流れが見えたら図式化し、「最初に何で興味を引き」「どこで売り」「不安をどう潰しているか」を指摘する。

### 7. 「なぜ今売れているのか」と「自分ならどう売るか」
最後に必ず2つ書いて締める：

- **なぜ今か（背景を1行）**：AI普及／副業解禁／物価高／人手不足／採用難など、表面でなく背景を見る
- **自分ならどう売るか（1行）**：ユーザー自身の言葉で書いてもらう（Claudeは叩き台を出すだけ）

### 8. ログを出力する
`/mnt/user-data/outputs/商品分解_YYYY-MM-DD.md` に保存して `present_files` で渡す。

## 注意

- 1日1商品で十分。分解の深さより「自分ならどう売るか」を毎日書くことが目的。
- 5個たまったら（週次）、共通点を3行でまとめる横断コメントを添える。
