# 財務会計の体系（桜井久勝『財務会計講義』第26版ベース）

## 財務会計の概念フレームワーク

### 財務報告の目的

投資家の意思決定に有用な情報（将来キャッシュフローの予測）を提供すること。

### 質的特性

| 特性 | 内容 |
|------|------|
| 意思決定有用性 | 上位概念。以下の特性で支えられる |
| 意思決定との関連性 | 情報価値の存在・フィードバック価値 |
| 信頼性 | 中立性・検証可能性・表現の忠実性 |
| 比較可能性 | 企業間・期間比較の可能性 |
| 理解可能性 | 合理的判断能力を持つ利用者が理解可能 |

### 認識と測定

- **発生主義**: 現金収支ではなく経済的事象の発生時に認識
- **実現主義**: 収益は実現時（財・サービスの引渡し＋対価の受領）
- **費用収益対応の原則**: 収益に対応する費用を同一期間に計上

## 財務諸表の体系

### 貸借対照表（B/S）

```
資産 = 負債 + 純資産

【資産の分類】
流動資産: 正常営業循環基準 + 1年基準
  ├ 当座資産（現金、売掛金、有価証券）
  ├ 棚卸資産（商品、製品、仕掛品、原材料）
  └ その他流動資産（前払費用、短期貸付金）
固定資産:
  ├ 有形固定資産（土地、建物、機械装置）
  ├ 無形固定資産（のれん、ソフトウェア、特許権）
  └ 投資その他の資産（投資有価証券、長期貸付金）
繰延資産: 株式交付費、社債発行費、開業費等

【負債の分類】
流動負債: 買掛金、短期借入金、未払法人税等、契約負債
固定負債: 社債、長期借入金、退職給付引当金、資産除去債務

【純資産の構成】
株主資本: 資本金 + 資本剰余金 + 利益剰余金 - 自己株式
その他の包括利益累計額: 評価差額金、繰延ヘッジ損益、為替換算調整勘定、退職給付調整累計額
新株予約権
非支配株主持分（連結のみ）
```

### 損益計算書（P/L）

```
売上高
- 売上原価
= 売上総利益（粗利）
- 販売費及び一般管理費
= 営業利益
+ 営業外収益 - 営業外費用
= 経常利益
+ 特別利益 - 特別損失
= 税引前当期純利益
- 法人税等（法人税、住民税、事業税）
± 法人税等調整額
= 当期純利益
```

### 包括利益計算書

```
当期純利益
+ その他の包括利益（OCI）
  ├ その他有価証券評価差額金
  ├ 繰延ヘッジ損益
  ├ 為替換算調整勘定
  ├ 退職給付に係る調整額
  └ 持分法適用会社のOCI
= 包括利益
```

### キャッシュ・フロー計算書（C/F）

| 区分 | 内容 | 健全な兆候 |
|------|------|-----------|
| 営業CF | 本業による現金創出 | プラスが基本 |
| 投資CF | 設備投資・M&A等 | 成長期はマイナス |
| 財務CF | 借入・返済・配当 | 安定期はマイナス |
| FCF | 営業CF + 投資CF | プラスが望ましい |

**間接法のポイント**: 税引前利益から非資金項目（減価償却費、引当金増減）と運転資本増減を調整

## 資産の会計処理

### 金融商品会計（企業会計基準第10号）

| 分類 | 評価 | 評価差額 |
|------|------|---------|
| 売買目的有価証券 | 時価 | P/L（営業外損益） |
| 満期保有目的債券 | 償却原価 | ー |
| 子会社・関連会社株式 | 取得原価 | ー（減損あり） |
| その他有価証券 | 時価 | OCI（純資産直入） |

**減損処理**: 時価が取得原価の50%以上下落 → 原則として減損（回復可能性を考慮）

### 棚卸資産（企業会計基準第9号）

- 評価基準: 取得原価（低価法で正味売却価額まで切下げ）
- 評価方法: 個別法、先入先出法、総平均法、移動平均法、売価還元法
- **IT企業**: 仕掛品（受託開発の進行中プロジェクト）の評価が重要

### 固定資産と減価償却

| 方法 | 特徴 | 典型的な適用 |
|------|------|-------------|
| 定額法 | 毎期均等額 | 建物、無形固定資産 |
| 定率法 | 初期に多く計上 | 機械装置、器具備品 |
| 生産高比例法 | 利用度に比例 | 鉱業用設備等 |

**ソフトウェア（実務指針第12号）**:
- 受注制作: 個別原価計算（請負は進行基準も）
- 自社利用: 将来の収益獲得・費用削減が確実 → 無形固定資産（3〜5年均等償却）
- 市場販売目的: 製品マスター完成まで研究開発費、以後は無形固定資産

### 減損会計（企業会計基準適用指針第6号）

```
Step 1: 減損の兆候（営業損益マイナス、市場価格の著しい下落等）
Step 2: 減損損失の認識判定（割引前CF < 帳簿価額 → 減損）
Step 3: 減損損失の測定（帳簿価額 - 回収可能価額）
  回収可能価額 = max(正味売却価額, 使用価値)
```

**のれんの減損**: 連結のれんは20年以内で規則的に償却（J-GAAP）+ 減損テスト

### リース会計（企業会計基準第13号 → 新基準: 2027年4月〜）

**現行基準**:
- ファイナンスリース: 売買処理（資産・負債計上）
- オペレーティングリース: 賃貸借処理

**新リース会計基準（2027年4月〜強制適用）**:
- 借手: 原則すべてのリースで使用権資産・リース負債を認識（IFRS16に近似）
- 短期リース（12ヶ月以下）・少額リースは免除可能
- 影響: オフバランスのオペリースがオンバランス化 → 総資産増・自己資本比率低下

### 退職給付会計（企業会計基準第26号）

```
退職給付引当金 = 退職給付債務(PBO) - 年金資産の公正価値
  ± 未認識数理計算上の差異
  ± 未認識過去勤務費用
```

- 割引率: 安全性の高い長期債券の利回り
- 期待運用収益率: 年金資産のポートフォリオに基づく
- **数理計算上の差異**: 発生年度に一括 or 平均残存勤務期間で償却

## 負債・純資産の会計

### 引当金の4要件

1. 将来の特定の費用または損失
2. 発生が当期以前の事象に起因
3. 発生の可能性が高い
4. 金額を合理的に見積もれる

### 自己株式

- 取得: 純資産の控除項目（資産ではない）
- 処分: 自己株式処分差額 → その他資本剰余金
- 消却: 自己株式とその他資本剰余金を減額

### ストック・オプション（企業会計基準第8号）

- 付与日の公正価値でオプション価値を算定
- 権利確定期間にわたり費用計上（株式報酬費用 / 新株予約権）
- 権利行使時: 新株予約権 → 資本金・資本準備金

## 連結会計

### 連結の範囲

- **子会社**: 議決権の過半数を所有（実質支配力基準）
- **関連会社**: 議決権の20%以上（重要な影響力基準）→ 持分法

### 連結手続

```
1. 投資と資本の相殺消去（のれん発生）
2. 債権債務の相殺消去
3. 取引高の相殺消去
4. 未実現利益の消去（ダウンストリーム/アップストリーム）
5. 非支配株主持分の計上
6. 持分法の適用（関連会社）
```

### のれん

- **J-GAAP**: 20年以内で規則的に償却
- **IFRS**: 非償却、毎期減損テスト
- **負ののれん**: 発生年度に一括利益計上

## 税効果会計（企業会計基準第28号）

### 一時差異

| 種類 | 内容 | 税効果 |
|------|------|--------|
| 将来減算一時差異 | 将来の課税所得を減額 | 繰延税金資産（DTA） |
| 将来加算一時差異 | 将来の課税所得を増額 | 繰延税金負債（DTL） |

**典型的なDTA**: 貸倒引当金限度超過額、賞与引当金、減損損失、繰越欠損金
**典型的なDTL**: その他有価証券評価差額金（評価益側）、圧縮記帳

### 回収可能性の判定（適用指針第26号）

| 分類 | 収益力 | 回収可能性 |
|------|--------|-----------|
| 分類1 | 安定的に課税所得発生 | 全額回収可能 |
| 分類2 | 安定的だが変動あり | スケジューリング可能な範囲 |
| 分類3 | 課税所得はあるが不安定 | 将来5年の合理的見積り |
| 分類4 | 重要な税務上の欠損金 | 翌期の課税所得見積り |
| 分類5 | 過去も将来も欠損 | 原則として認められない |

## 1株当たり情報

```
基本的EPS = 当期純利益(親会社株主帰属) ÷ 期中平均株式数
希薄化後EPS = (純利益 + 希薄化調整) ÷ (平均株式数 + 潜在株式数)
```

- 自己株式は控除
- ストック・オプション、転換社債による希薄化を考慮
