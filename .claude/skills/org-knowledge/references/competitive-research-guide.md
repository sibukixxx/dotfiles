# 競合調査ガイド

## 調査プロセス

```
1. 競合特定 → 2. 情報収集 → 3. 分析 → 4. 比較表作成 → 5. 戦略示唆
```

## 1. 競合の特定

### 3層で分類
- **直接競合**: 同じ顧客・同じソリューション
- **間接競合**: 同じ顧客・異なるアプローチ
- **潜在競合**: 異なる市場から参入の可能性

### 情報源
- WebSearch（「[サービスカテゴリ] 比較」「[競合名] 評判」）
- 求人情報（組織体制・注力領域の推測）
- IR情報・プレスリリース
- SNS（経営者・社員の発信）
- 展示会・カンファレンス情報

## 2. 収集する情報

| カテゴリ | 項目 |
|---------|------|
| 基本情報 | 設立年、社員数、資本金、拠点 |
| サービス | サービス一覧、価格帯、ターゲット |
| 強み | 技術力、顧客基盤、ブランド、特許 |
| 弱み | 組織の課題、顧客の不満（レビューサイト） |
| 戦略 | 最近のプレスリリース、採用動向、投資先 |
| 財務 | 売上、利益率（上場企業のみ） |

## 3. 分析フレームワーク

### ポジショニングマップ
2軸で市場における位置関係を可視化:
- 例: 横軸=価格（低←→高）、縦軸=カスタマイズ性（標準←→オーダーメイド）
