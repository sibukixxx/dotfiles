# MBOの法務と税務ガイド

「MBOの法務と税務」（長島・大野・常松法律事務所 / デロイト トーマツ税理士法人）に基づく。

## MBOとは

**MBO（Management Buyout）**: 経営陣が自社の株式を取得し、オーナーとなる取引。

### 典型的なMBOの目的
- 上場企業の非公開化（ゴーイング・プライベート）
- オーナー企業の事業承継
- 親会社からの独立（カーブアウト）
- 経営陣のインセンティブ強化

## MBOのストラクチャー

### 基本スキーム

```
Step 1: SPC（特別目的会社）設立
  経営陣（+ PE/金融機関）が出資

Step 2: SPCが資金調達
  LBO（レバレッジド・バイアウト）ローン

Step 3: SPCが対象会社の株式を取得
  TOB（公開買付け）/ 相対取引

Step 4: スクイーズアウト（少数株主の締出し）
  株式等売渡請求 / 株式併合

Step 5: SPCと対象会社の合併
  LBOローンの返済原資を対象会社のCFに
```

### ストラクチャーの選択肢

| 手法 | 概要 | 適用場面 |
|------|------|---------|
| **TOB + スクイーズアウト** | 公開買付け→残余株主締出し | 上場企業のMBO |
| **相対取得** | 大株主から直接取得 | 非上場企業、少数株主なし |
| **株式交換** | SPC株式と対象会社株式を交換 | 税制適格の要件を満たす場合 |
| **第三者割当増資** | 対象会社がSPCに新株発行 | 希薄化で既存株主を排除 |

## 法務上の論点

### 1. 取締役の善管注意義務・忠実義務

MBOでは経営陣が **買主と売主の双方** に立つ利益相反:

```
経営陣の立場:
  買主として → できるだけ安く買いたい
  取締役として → 株主利益を最大化すべき（善管注意義務）
```

**対応策**:
- **特別委員会** の設置（独立社外取締役・外部有識者）
- **独立した第三者評価機関** による株式価値算定
- **マーケットチェック**: 他の買い手候補の検討
- **マジョリティ・オブ・マイノリティ条件**: 少数株主の過半数の賛同

### 2. 公開買付け（TOB）

#### TOB規制の概要
- **強制TOB**: 市場外で1/3超取得する場合はTOB必須
- **全株TOB**: MBOでは全株式を対象とするTOBが一般的
- **買付期間**: 20〜60営業日
- **撤回条件**: 予め公開買付届出書に記載した条件のみ

#### TOB価格の決定
- DCF法、類似会社比較法、市場株価法を併用
- **プレミアム**: 市場株価の30-50%が一般的
- **フェアネス・オピニオン**: 独立した第三者の意見書

### 3. スクイーズアウト（少数株主の締出し）

| 手法 | 要件 | メリット | デメリット |
|------|------|---------|----------|
| 株式等売渡請求 | 90%以上保有 | 迅速（株主総会不要） | 仮処分リスク |
| 株式併合 | 2/3以上の賛成 | 90%未満でも可能 | 株主総会が必要 |

### 4. 反対株主の株式買取請求権

- スクイーズアウト時に反対株主が「公正な価格」での買取を請求可能
- **価格決定の裁判**: 当事者間で合意できない場合は裁判所が決定
- **ナカリセバ価格**: MBOがなかったら株式が有していたであろう価格

### 5. 情報開示

- 意見表明報告書（対象会社の取締役会の意見）
- 特別委員会の答申内容
- 株式価値算定書の概要
- 利益相反回避措置の内容

## 税務上の論点

### 1. SPCの税務

#### LBOローンの利息の損金算入
- 合併後、対象会社のCFでLBOローンを返済
- ローン利息は対象会社の損金に算入可能
- **過大支払利子税制**: 対象純支払利子等 > 調整所得金額の20% → 超過分は損金不算入

#### 過大支払利子税制の計算

```
調整所得金額 = 所得金額 + 対象純支払利子等 + 減価償却費
損金算入限度額 = 調整所得金額 × 20%
超過額 = 対象純支払利子等 - 損金算入限度額 → 損金不算入
（超過額は7年間繰越可能）
```

### 2. 株主側の税務

#### 個人株主
- TOBで株式を売却 → **譲渡所得**（申告分離課税: 20.315%）
- みなし配当が生じる場合 → **配当所得**（総合課税 or 申告分離）

#### 法人株主
- 譲渡益 → 法人税課税
- みなし配当 → 受取配当等の益金不算入の適用可否を検討

### 3. みなし配当

スクイーズアウト時に発生する可能性:

```
交付金銭 - 資本金等の額に対応する部分 = みなし配当

例: 1株500円で取得、資本金等対応額が200円の場合
  みなし配当: 500 - 200 = 300円/株
  譲渡損益: 200 - 取得価額
```

### 4. 合併の税務

#### 適格合併の要件（SPC ← 対象会社）

MBO後のSPCと対象会社の合併は **非適格合併** になるケースが多い:
- 金銭を対価としたMBO → 共同事業要件を満たさない
- 非適格合併 → 対象会社の資産を時価評価 → 含み益に課税

**対策**: SPCを吸収合併存続会社とし、対象会社を消滅会社にするケースが一般的

### 5. 繰越欠損金の取扱い

- 非適格合併: 被合併法人の繰越欠損金は引き継げない
- 適格合併: 引き継ぎ可能だが **みなし共同事業要件** を満たす必要あり
- **特定資本関係5年要件**: 5年以内の資本関係 → 欠損金に制限

### 6. グループ通算制度との関係

- MBO後にグループ通算を適用する場合:
  - 加入時の時価評価の要否を検討
  - 繰越欠損金の持ち込み制限
  - 詳細は `group-tax-consolidation.md` を参照

## 非上場会社のMBO

### 上場会社との違い

| 項目 | 上場会社 | 非上場会社 |
|------|---------|----------|
| 株式取得方法 | TOB | 相対取引 |
| 価格決定 | 市場株価+プレミアム | **DCF/純資産法** |
| スクイーズアウト | 必要なケースが多い | 少数株主がいなければ不要 |
| 規制 | 金商法（TOB規制） | 会社法のみ |
| 資金調達 | LBOローン（大規模） | 自己資金+銀行借入 |

### 事業承継型MBO

```
現オーナー → 後継経営者（MBO）
  株式譲渡（相対取引）
  資金: 銀行借入 + PE出資
  税務: 譲渡所得（個人）/ みなし配当の検討
```

### 株式評価（非上場）

| 方法 | 概要 | 適用場面 |
|------|------|---------|
| DCF法 | 将来CFの現在価値 | 最も一般的 |
| 類似会社比較法 | 類似上場会社のマルチプルを適用 | DCFの補完 |
| 純資産法 | 資産の時価合計 - 負債 | 資産保有型会社 |
| 配当還元法 | 将来配当の現在価値 | 税務上の少数株主評価 |
