## 企業会計原則（一般原則）

| # | 原則 | 内容 |
|---|------|------|
| 1 | 真実性の原則 | 企業の財政状態・経営成績に関して真実な報告を提供 |
| 2 | 正規の簿記の原則 | 正確な会計帳簿を作成（網羅性・検証可能性・秩序性） |
| 3 | 資本取引・損益取引区分の原則 | 資本取引と損益取引を明瞭に区別 |
| 4 | 明瞭性の原則 | 利害関係者に必要な情報を明瞭に表示 |
| 5 | 継続性の原則 | 会計処理の方法を毎期継続して適用 |
| 6 | 保守主義の原則 | 予測される将来の危険に備えて慎重な判断に基づく会計処理 |
| 7 | 単一性の原則 | 信頼しうる会計記録に基づき、目的に応じ形式が異なっても実質は統一 |

## 会社法の計算規定

### 分配可能額

```
分配可能額 = その他資本剰余金 + その他利益剰余金
           - 自己株式の帳簿価額
           - 当期に処分した自己株式の対価
           ± その他の調整項目

※のれん等調整額の控除:
  のれん × 1/2 + 繰延資産 > その他資本剰余金 + その他利益剰余金 の場合に控除
```

### 計算書類の体系

| 書類 | 法律 | 提出先 |
|------|------|--------|
| 計算書類（B/S, P/L, 株主資本等変動計算書, 個別注記表） | 会社法 | 株主総会 |
| 有価証券報告書（連結F/S, 個別F/S含む） | 金商法 | EDINET |
| 四半期報告書 | 金商法 | EDINET |
| 確定申告書（別表） | 法人税法 | 税務署 |

## 金融商品取引法の開示

### 有価証券報告書の記載事項

```
第一部 企業情報
  第1 企業の概況
    1. 主要な経営指標等の推移（5期分）
    2. 沿革
    3. 事業の内容
    4. 関係会社の状況
    5. 従業員の状況
  第2 事業の状況
    1. 経営方針、経営環境及び対処すべき課題等
    2. サステナビリティに関する考え方及び取組
    3. 事業等のリスク
    4. 経営者による財政状態、経営成績及びキャッシュ・フローの状況の分析（MD&A）
    5. 経営上の重要な契約等
    6. 研究開発活動
  第3 設備の状況
  第4 提出会社の状況（株式、配当、コーポレートガバナンス）
  第5 経理の状況（連結F/S + 個別F/S + 注記）
  第6 提出会社の株式事務の概要
  第7 提出会社の参考情報
第二部 提出会社の保証会社等の情報
```

### KAM（監査上の主要な検討事項）

2021年3月期から強制適用:
- 監査人が特に重要と判断した事項を監査報告書に記載
- のれんの減損、収益認識、税効果の回収可能性が頻出
- 投資家と監査人の情報ギャップを埋める目的

## 法人税法との主要差異（申告調整）

### 加算調整（会計上の費用 > 税務上の損金）

| 項目 | 内容 |
|------|------|
| 交際費 | 資本金1億円超: 飲食費50%まで損金算入、それ以外は全額損金不算入 |
| 寄附金 | 損金算入限度額超過部分 |
| 減価償却 | 償却限度額超過（会計の耐用年数 < 税法の耐用年数の場合） |
| 貸倒引当金 | 一般引当は中小法人以外は損金不算入 |
| 賞与引当金 | 損金不算入（支払時に損金算入） |
| 退職給付引当金 | 損金不算入（掛金拠出・退職金支払時に損金算入） |
| 役員給与 | 不相当に高額な部分、事前確定届出がない賞与 |

### 減算調整（会計上の収益 < 税務上の益金）

| 項目 | 内容 |
|------|------|
| 受取配当金 | 完全子法人: 全額益金不算入、関連法人: 一部益金不算入 |
| 所得税額控除 | 源泉所得税の税額控除 |

### 繰越欠損金

- 控除限度: 所得金額の50%（中小法人は100%）
- 繰越期間: 10年
- 青色申告法人のみ
- グループ通算制度下での制限あり（→ group-tax-consolidation.md 参照）
