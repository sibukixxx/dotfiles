# グループ通算制度ガイド

「詳解 グループ通算制度 Q&A 第3版」（デロイト トーマツ）に基づく。

## 制度概要

2022年4月1日以後開始事業年度から適用。旧連結納税制度を廃止し、グループ通算制度に移行。

### 旧連結納税制度との主な違い

| 項目 | 旧連結納税 | グループ通算 |
|------|----------|------------|
| 申告方式 | 親法人が一体申告 | **各法人が個別申告** |
| 損益通算 | グループ全体で通算 | グループ全体で通算（維持） |
| 修更正 | グループ全体に波及 | **原則、当該法人のみ** |
| 税効果会計 | 複雑 | **簡素化** |
| 地方税 | 連結法人税個別帰属額 | 各法人の法人税額に基づく |

## 適用要件

### 通算親法人の要件
- 内国法人であること
- 普通法人または協同組合等であること
- 他の通算グループに属していないこと

### 通算子法人の要件
- 通算親法人との間に **完全支配関係**（100%保有）があること
- 内国法人であること

### 完全支配関係の判定
```
親法人 ─ 100% ─→ 子法人A ← 通算対象
         100% ─→ 子法人B ← 通算対象
                   50% ─→ 孫法人C ← 対象外（100%でない）
```

**間接保有も含む**: 親→子A(100%)→孫D(100%) → 孫Dも通算対象

## 適用開始・加入

### 適用開始
1. 親法人が「通算承認申請書」を所轄税務署に提出
2. 提出期限: 適用開始事業年度開始の日の **3ヶ月前の日**
3. みなし承認: 開始日前日までに却下通知がなければ承認とみなす

### 子法人の加入
- **設立加入**: 親法人が設立した100%子法人 → 設立日から自動加入
- **買収加入**: 100%取得した子法人 → 翌事業年度から加入

### 加入時の時価評価

原則として加入法人の資産を **時価評価** し、評価損益を計上:

| 資産 | 時価評価対象 |
|------|------------|
| 固定資産 | 帳簿価額1,000万円以上 |
| 有価証券 | 帳簿価額1,000万円以上 |
| 金銭債権 | 帳簿価額1,000万円以上 |
| 繰延資産 | 帳簿価額1,000万円以上 |

**時価評価不要の例外**:
- 適用開始前から完全支配関係が継続している法人
- 共同事業に係る要件を満たす法人

### 加入時の繰越欠損金の制限

原則: 加入法人の繰越欠損金は **切り捨て**
例外: 適用開始前から完全支配関係が継続 → 持ち込み可能（SRLY制限あり）

## 離脱・取止め

### 離脱
- 完全支配関係が崩れた場合（株式売却等）→ 離脱
- 離脱法人は **時価評価** が必要（一定の例外あり）

### 取止め
- グループ通算の取止め → 親法人が届出
- **5年間** 再適用不可（やむを得ない事由を除く）
- 取止め時も各法人で時価評価が必要

## 損益通算の仕組み

### 基本的な流れ

```
Step 1: 各法人が個別に所得/欠損を計算
Step 2: 欠損法人の欠損金をグループ内で損益通算
Step 3: 通算後の所得に基づき各法人が法人税を計算
Step 4: 各法人が個別に申告・納付
```

### 損益通算の計算

```
通算前所得: A社 +100、B社 +50、C社 -80
通算欠損: -80をA社・B社の所得比率で配分
  A社: 100/(100+50) × (-80) = -53.3
  B社: 50/(100+50) × (-80) = -26.7
通算後所得: A社 46.7、B社 23.3、C社 0
```

### 遮断措置（修更正時）

修更正があった場合、原則として **当該法人のみ** で完結:
- 他の法人の所得・税額には影響しない（遮断）
- 旧連結納税の最大の問題点を解消

## 各種税額控除

### グループ通算で通算される主な制度

| 制度 | 通算方法 |
|------|---------|
| 外国税額控除 | グループ全体で限度額を計算、各法人に配分 |
| 研究開発税制 | グループ全体の試験研究費で税額控除を計算 |
| 所得税額控除 | 各法人で個別に計算 |
| 受取配当等の益金不算入 | 関連法人株式の判定はグループベース |

### 研究開発税制のグループ通算

```
グループ全体の試験研究費 → 控除率を算出 → 各法人の試験研究費に控除率を適用
```

メリット: 試験研究費の規模が大きくなり、控除率が上がる場合がある

## 税効果会計

### 繰延税金資産の回収可能性

グループ通算制度下では:
1. **損益通算** による将来減算一時差異の回収可能性を考慮
2. 各法人の **分類判定**（企業会計基準適用指針第28号）は個別に行う
3. グループ通算による税効果は **税金費用の配分** で処理

### 仕訳例（損益通算の税効果）

```
【欠損法人C社】
借方: 未収還付法人税等  ¥XXX
貸方: 法人税等調整額    ¥XXX
摘要: グループ通算による損益通算の税効果

【所得法人A社】
借方: 法人税等調整額    ¥XXX
貸方: 未払法人税等      ¥XXX
摘要: グループ通算による損益通算の負担
```

## 実務上の検討ポイント

### グループ通算を採用すべきか

**メリット**:
- グループ内の損益通算（黒字・赤字の相殺）
- 研究開発税制の有利な適用
- 外国税額控除の有利な適用

**デメリット**:
- 管理コストの増加（各法人で申告）
- 加入・離脱時の時価評価
- 繰越欠損金の制限
- 5年間の再適用制限

### 判断フローチャート

```
グループに赤字法人がある？
├── Yes → 損益通算のメリットが大きい → 検討価値あり
└── No → 研究開発費がグループで大きい？
          ├── Yes → 税額控除のメリットを試算
          └── No → 管理コスト増のデメリットが上回る可能性
```
