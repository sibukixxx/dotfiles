# 提案書・見積書ガイド

## 提案書の基本構成

### 1. 表紙
- タイトル（「〇〇に関するご提案」）
- 提出日
- 提出先（御中）
- 自社名・ロゴ

### 2. エグゼクティブサマリー（1ページ）
- 課題の要約
- 提案の核心（一文で）
- 期待効果（数字で）

### 3. 課題の整理
- ヒアリング内容の要約
- 現状の課題（3〜5個に整理）
- 課題を放置した場合のリスク

### 4. 提案内容
- ソリューション概要
- アプローチ（フェーズ分け）
- 技術選定の根拠
- 差別化ポイント

### 5. 実績・事例
- 類似案件の成功事例（2〜3件）
- 定量的な成果（コスト削減率、効率化等）

### 6. 体制・スケジュール
- プロジェクト体制図
- 主要メンバーの経歴
- マイルストーン付きスケジュール

### 7. 費用
- 概算費用（フェーズ別）
- 詳細見積（別紙）
- 支払条件

### 8. リスクと対策
- 想定リスク（3つ以内）
- それぞれの対策
