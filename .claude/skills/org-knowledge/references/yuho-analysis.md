# 有価証券報告書（有報）分析ガイド

EY新日本「3つの視点で会社がわかる『有報』の読み方 第3版」に基づく。

## 3つの視点

### 視点1: 大局的視点（全体像を把握）

有報の全体構成を俯瞰し、会社の「顔」を掴む。

#### 有報の基本構成

| セクション | 主な記載内容 | 着目ポイント |
|-----------|------------|------------|
| 第一部 企業情報 | | |
| 第1 企業の概況 | 主要経営指標、沿革、事業内容 | 5期分の指標推移 |
| 第2 事業の状況 | 経営方針、MD&A、リスク情報 | **最重要セクション** |
| 第3 設備の状況 | 設備投資、主要設備 | 成長投資の方向性 |
| 第4 提出会社の状況 | 株式、配当、コーポレートガバナンス | 株主構成、役員報酬 |
| 第5 経理の状況 | 財務諸表、注記 | 会計方針、見積り |
| 第二部 提出会社の保証会社等の情報 | | |

#### 主要経営指標の等比較（5期分）

チェックすべき指標:
- 売上高の成長トレンド
- 営業利益率の推移
- 自己資本比率の安定性
- 1株当たり当期純利益（EPS）の成長
- 配当性向の一貫性

### 視点2: ストーリー別の視点（イベント分析）

M&A、業績悪化、構造改革等のイベントが有報にどう反映されるかを読み解く。

#### M&A発生時に見るべき箇所

```
1. 第1 企業の概況 → 関係会社の異動
2. 第2 事業の状況 → セグメント変更の有無
3. 第5 経理の状況 → のれん・無形資産の計上額
   - 注記: 企業結合
   - 注記: のれんの減損テスト
4. KAM → 監査上の主要な検討事項としてM&Aが取り上げられているか
```

#### 業績悪化時に見るべき箇所

```
1. MD&A → 業績悪化の要因分析（一過性 vs 構造的）
2. リスク情報 → 継続企業の前提に関する注記の有無
3. 減損損失 → 固定資産・のれんの減損テスト
4. 引当金 → 事業構造改善引当金、受注損失引当金
5. 後発事象 → 期末後の重要事象
```

#### 不正・不祥事発覚時

```
1. 訂正報告書の有無
2. 内部統制報告書 → 開示すべき重要な不備
3. 監査報告書 → 意見の種類（限定付適正/不適正/意見不表明）
4. KAM → 不正リスクへの監査対応
```

### 視点3: 項目別の視点（詳細分析）

個々の項目を深掘りして読み解く。

#### MD&A（経営者による財政状態、経営成績及びキャッシュ・フローの状況の分析）

最重要セクション。以下を確認:
- 経営成績の分析（増減要因を定量的に説明しているか）
- セグメント別の業績（成長/衰退セグメントの識別）
- キャッシュ・フローの状況（営業CFがプラスか）
- 資本の財源と流動性（借入余力、コミットメントライン）
- 経営上の重要な契約等

#### 会計上の見積り開示（見積開示会計基準）

2021年3月期から適用。以下を確認:
- **見積りの内容**: 何の見積りか
- **金額**: B/S計上額
- **会計上の見積りの内容に関する情報**:
  - 見積りの算出方法
  - 主要な仮定
  - 翌年度に与える影響

重要な見積り項目の例:
- のれんの評価（減損テスト）
- 固定資産の減損
- 繰延税金資産の回収可能性
- 退職給付債務
- 貸倒引当金

#### KAM（監査上の主要な検討事項）

2021年3月期から適用。監査人が特に重要と判断した事項:
- **なぜKAMに選定されたか**: リスクの所在
- **監査上の対応**: 監査人が実施した手続
- **会社の見積りとの整合性**: 楽観的すぎないか

KAMで頻出するテーマ:
1. のれん・無形資産の評価
2. 収益認識（特に長期契約、変動対価）
3. 固定資産の減損
4. 繰延税金資産の回収可能性
5. 引当金の十分性

#### セグメント情報

- 報告セグメントの定義と変更理由
- セグメント利益率の比較
- セグメント資産の配分
- 地域別売上高

#### IFRS適用企業の場合

日本基準との主な差異:
| 項目 | 日本基準 | IFRS |
|------|---------|------|
| のれん | 定額償却（最長20年） | 非償却（年次減損テスト） |
| 開発費 | 全額費用処理 | 要件を満たせば資産計上 |
| 退職給付 | 遅延認識あり | OCI即時認識 |
| リース | 所有権移転/ファイナンスリース | 使用権資産（原則全リース） |
| 収益認識 | ほぼ同一（IFRS15準拠） | 基準の源泉 |
