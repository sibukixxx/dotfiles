# ROIC経営・資本コスト管理ガイド

「ROIC超入門」（松永博樹・伊藤学）に基づく。

## ROICとは

**ROIC（Return On Invested Capital）= 投下資本利益率**

```
ROIC = NOPAT ÷ 投下資本

NOPAT（税引後営業利益）= 営業利益 ×（1 - 実効税率）
投下資本 = 有利子負債 + 株主資本（= 事業に投下された資本の合計）
```

## ROA・ROEとの違い

| 指標 | 分子 | 分母 | 視点 |
|------|------|------|------|
| **ROE** | 当期純利益 | 株主資本 | 株主から見た収益性 |
| **ROA** | 当期純利益（or 経常利益） | 総資産 | 資産全体の効率性 |
| **ROIC** | NOPAT | 投下資本 | **事業の本質的な稼ぐ力** |

### ROICが優れている理由

1. **財務レバレッジに影響されない**: ROEは借入で嵩上げ可能だがROICはできない
2. **特別損益を除外**: NOPATは本業の利益のみ
3. **事業に使った資本のみ**: 遊休資産や余剰現金を除外
4. **事業別に計算可能**: 事業ポートフォリオ管理に最適

## 資本コスト（WACC）

**WACC（加重平均資本コスト）**: 企業が資金調達に要するコスト

```
WACC = E/(D+E) × Re + D/(D+E) × Rd ×（1-t）

E: 株主資本の時価
D: 有利子負債
Re: 株主資本コスト（CAPM等で算出）
Rd: 負債コスト（借入金利）
t: 実効税率
```

### 株主資本コスト（CAPM）

```
Re = Rf + β ×（Rm - Rf）

Rf: リスクフリーレート（国債利回り、約0.5-1.0%）
β: 個別企業のベータ値
Rm - Rf: マーケットリスクプレミアム（約5-7%）
```

### ROIC > WACC の意味

```
ROIC > WACC → 価値創造（EVAスプレッドがプラス）
ROIC = WACC → 価値中立
ROIC < WACC → 価値破壊（資本コストを賄えていない）
```

**東証の要請**: PBR 1倍割れ企業は「資本コストや株価を意識した経営の実現に向けた対応」が求められている。

## ROICツリー（分解）

ROICを要素分解し、改善ドライバーを特定する:

```
ROIC
├── NOPAT マージン（= NOPAT ÷ 売上高）
│   ├── 売上総利益率
│   │   ├── 単価
│   │   └── 原価率
│   ├── 販管費率
│   │   ├── 人件費率
│   │   ├── 広告宣伝費率
│   │   └── その他経費率
│   └── 実効税率
│
└── 投下資本回転率（= 売上高 ÷ 投下資本）
    ├── 運転資本回転率
    │   ├── 売上債権回転期間（DSO）
    │   ├── 棚卸資産回転期間（DIO）
    │   └── 仕入債務回転期間（DPO）
    └── 固定資産回転率
        ├── 有形固定資産回転率
        └── 無形固定資産回転率
```

### CCC（キャッシュ・コンバージョン・サイクル）

```
CCC = DSO + DIO - DPO

DSO（売上債権回転期間）: 売掛金の回収にかかる日数
DIO（棚卸資産回転期間）: 在庫の滞留日数
DPO（仕入債務回転期間）: 買掛金の支払いまでの日数
```

CCCが短いほど資金効率が良い。

## 事業別ROIC管理

### 事業ポートフォリオ分析

4象限マトリクス:

```
         高ROIC
           │
    Stars  │  ?
   成長投資 │ 要判断
 ──────────┼──────────
    Cash   │  Dogs
   維持/配当│ 撤退/再建
           │
         低ROIC
  高成長率        低成長率
```

### 事業別ROIC算出のポイント

1. **本社費の配賦**: 売上高比率 or 従業員数比率で配賦
2. **共通資産の配分**: 使用面積比率 or 利用実績で配分
3. **のれんの取扱い**: のれん含む/含まないの両方で計算
4. **移転価格**: グループ内取引は市場価格で評価

## IT/SES企業のROIC分析

### IT企業特有の指標

| 指標 | 計算式 | 意味 |
|------|--------|------|
| エンジニア1人当たり売上 | 売上 ÷ エンジニア数 | 生産性 |
| エンジニア1人当たり粗利 | 粗利 ÷ エンジニア数 | 付加価値 |
| 稼働率 | 稼働人月 ÷ 在籍人月 | 資産効率 |
| 平均単価 | SES売上 ÷ 稼働人月 | 単価水準 |
| 採用ROI | 粗利増分 ÷ 採用コスト | 投資回収 |

### SES事業のROIC改善レバー

```
ROIC改善
├── マージン改善
│   ├── 平均単価の引き上げ（スキルアップ、上流案件）
│   ├── 原価率の低減（福利厚生の最適化）
│   └── 販管費の効率化（営業1人当たり管理人数）
│
└── 回転率改善
    ├── 稼働率の向上（待機期間の短縮）
    ├── 売上債権の早期回収（請求サイクルの短縮）
    └── 不採算案件の撤退（低単価案件の入れ替え）
```

## EVA（経済的付加価値）

```
EVA = NOPAT - 投下資本 × WACC
    = 投下資本 ×（ROIC - WACC）
```

EVAがプラスなら企業価値を創造している。

## 実務での活用ステップ

1. **現状把握**: 全社ROICとWACCを算出
2. **事業別分解**: 事業セグメント別にROICを算出
3. **ツリー分析**: ROICツリーでボトルネックを特定
4. **目標設定**: WACC+α をROIC目標に設定
5. **KPI連動**: ROICツリーの末端指標を現場KPIに落とし込む
6. **モニタリング**: 四半期ごとに進捗確認
