# エグゼクティブスタッフ手引き

## 1. 役割

- CEOの代わりに意思決定するのではなく、意思決定の質と速度を上げる
- 情報の洪水を構造化し、論点を減らす
- 会議を「話した」で終わらせず、決定とアクションに変える

## 2. CEO向けメモの型

1. 結論
2. なぜ今か
3. 選択肢
4. 推奨案
5. 主要リスク
6. 次のアクション

## 3. 重要案件のさばき方

### まず確認すること

- これは何を決める会話か
- 誰が決めるのか
- いつまでに決める必要があるか
- 不足情報は何か

### エスカレーション基準

- 会社の評判に影響する
- 既存方針を覆す
- 大口顧客 / 大型投資 / 主要人材に関わる
- 法務・財務・セキュリティ上の重大リスクがある

## 4. 会議運営

- 議題ごとに目的を `共有 / 議論 / 決定` で明示する
- 終了時に決定事項、保留事項、担当者、期限を固定する
- 次回会議で扱う前提条件も残す

## 5. 経営タスク管理

- 緊急度だけでなく重要度で並べる
- CEOしかできない仕事と委譲可能な仕事を分ける
- 依存関係があるタスクは束で管理する

## 6. アンチパターン

- 論点を増やすだけで結論を出さない
- 重要度の違う案件を同じ粒度で扱う
- 会議体を増やしてオーナーを曖昧にする
- CEO向けメモに生データをそのまま貼る
