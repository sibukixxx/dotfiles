## 1. キャッシュマネジメント

### 資金繰り管理

```
【日次資金繰り表】
  前日残高
  + 当日入金（売掛金回収、借入実行等）
  - 当日支出（仕入代金、人件費、借入返済等）
  = 当日残高

【月次資金繰り表】
  前月末残高
  + 経常収入（売上入金、受取利息等）
  - 経常支出（仕入、人件費、経費等）
  = 経常収支
  + 経常外収入（資産売却、借入等）
  - 経常外支出（設備投資、借入返済、配当等）
  = 当月末残高
```

### 運転資本管理（Working Capital）

```
運転資本 = 売上債権 + 棚卸資産 - 仕入債務

CCC（キャッシュ・コンバージョン・サイクル）
  = 売上債権回転日数 + 棚卸資産回転日数 - 仕入債務回転日数

SES企業の場合:
  売上債権回転日数: 月末締め翌月末払い → 約45-60日
  棚卸資産: ほぼゼロ（人的サービス）
  仕入債務回転日数: BP支払い → 約30-45日
  → CCC = 45 + 0 - 30 = 約15日
```

**改善施策**:
- 請求サイクルの短縮（月中締め、早期請求）
- ファクタリング活用（売掛金の早期現金化）
- BP支払条件の交渉（支払サイトの延長は慎重に）
- 大型案件の中間請求・マイルストーン請求

### 手元流動性の目安

```
手元流動性 = 現金預金 + すぐに換金できる有価証券

目安: 月商の1-3ヶ月分
  安定企業: 1ヶ月分
  成長企業: 2-3ヶ月分
  上場準備中: 3ヶ月分以上（IPO費用考慮）
```

## 2. 財務戦略・資金調達

### 最適資本構成

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-T)

Re: 株主資本コスト（CAPM: Rf + β × MRP）
Rd: 負債コスト（借入金利）
T: 実効税率
E: 時価ベース株主資本
D: 有利子負債

→ WACCを最小化する D/E比率を目指す
→ ただし財務安全性とのバランスが重要
```

### 資金調達手段

| 手段 | メリット | デメリット | 適用場面 |
|------|---------|-----------|---------|
| 銀行借入 | 機動的、希薄化なし | 返済義務、担保・保証 | 運転資金、設備投資 |
| 社債 | 大規模調達可、金利固定 | 格付け必要 | 中長期の大型投資 |
| 増資（公募・第三者割当） | 返済不要 | 希薄化、時間がかかる | 成長投資、財務改善 |
| 転換社債（CB） | 低金利、希薄化遅延 | 転換時に希薄化 | 成長期の調達 |
| ベンチャーデット | 希薄化限定的 | 金利高め、ワラント | スタートアップの追加調達 |
| リース・割賦 | オフバランス効果（旧基準） | 総コスト高 | 設備投資 |

### 銀行交渉のポイント

- **情報の非対称性を減らす**: 月次試算表、事業計画を定期共有
- **メインバンク制**: 取引銀行の序列を意識した借入配分
- **コベナンツ管理**: 財務制限条項（自己資本比率、DSCR等）の遵守確認
- **金利交渉**: TIBOR連動、スプレッド交渉、借換え時の競合見積もり

## 3. 経営管理（FP&A）

### 予算策定プロセス

```
【年次予算の策定フロー】
1. 経営方針・目標の提示（CEO/CFO → 全部門）
2. ボトムアップ予算の積み上げ（各部門）
3. トップダウンとの調整・すり合わせ
4. 取締役会承認
5. 月次モニタリング → 四半期フォーキャスト更新

【SES企業の予算構造】
売上 = エンジニア数 × 稼働率 × 平均単価 × 営業日数
売上原価 = エンジニア人件費 + BP費用
粗利 = 売上 - 売上原価
→ 稼働率と単価が最重要KPI
```

### 予実分析のフレームワーク

```
差異分析（Variance Analysis）:
  売上差異
  ├ 数量差異: (実績数量 - 予算数量) × 予算単価
  └ 価格差異: (実績単価 - 予算単価) × 実績数量

SES企業の例:
  予算: 100人 × 90% × ¥80万 = ¥7,200万
  実績: 105人 × 88% × ¥82万 = ¥7,577万
  差異: +¥377万
    人数効果: +5人 × 90% × ¥80万 = +¥360万
    稼働率効果: 105人 × (-2%) × ¥80万 = -¥168万
    単価効果: 105人 × 88% × (+¥2万) = +¥185万
```

### KPIダッシュボード設計

| カテゴリ | KPI | 頻度 | 責任者 |
|---------|-----|------|--------|
| 売上 | MRR/ARR、新規受注額 | 月次 | 営業部長 |
| 効率 | エンジニア稼働率 | 週次 | 事業部長 |
| 収益性 | 粗利率、営業利益率、ROIC | 月次 | CFO |
| 資本効率 | CCC、運転資本回転率 | 月次 | 経理部長 |
| 成長 | 売上成長率、従業員数 | 月次 | CEO |
| 健全性 | 自己資本比率、手元流動性 | 月次 | CFO |

## 4. IR・資本市場対応

### 決算説明資料の構成

```
1. エグゼクティブサマリー（ハイライト3-5点）
2. 業績概要（P/L、B/S、CF要約）
3. セグメント別業績
4. トピックス（新規事業、M&A等）
5. 業績予想と前提条件
6. 株主還元方針
7. 中期経営計画の進捗
8. ESG/サステナビリティ
```

### 株主還元政策

```
総還元性向 = (配当 + 自社株買い) ÷ 当期純利益

【配当政策の類型】
安定配当: 業績に関わらず一定額（信頼性重視）
配当性向: 利益の一定割合（連動性重視）
DOE: 純資産配当率（安定性と資本効率の両立）
  DOE = 配当 ÷ 純資産 = 配当性向 × ROE

プログレッシブ配当: 減配せず段階的に増配
```

### PBR 1倍割れ対策（東証要請）

```
PBR = 株価 ÷ BPS = ROE × PER

PBR < 1倍の原因:
  ROEが低い → 資本効率改善（ROIC経営、事業ポートフォリオ見直し）
  PERが低い → 成長期待の醸成（IR強化、中計の実行力）

施策:
  ├ ROE向上: 自社株買い、政策保有株式の売却、低収益事業の撤退
  ├ 成長投資: M&A、R&D、人的資本投資
  └ 対話強化: 資本コスト経営の開示、事業ポートフォリオの説明
```

## 5. IPO準備

### IPO準備のタイムライン

```
N-3期: 管理体制構築開始
  - 監査法人ショートレビュー
  - 内部統制の基盤整備
  - 会計方針の整理

N-2期: 本格準備
  - 監査法人による会計監査開始
  - 主幹事証券会社の選定
  - 予算管理体制の確立
  - 関連当事者取引の整理

N-1期: 直前期
  - 内部統制の有効性評価
  - 上場審査書類の準備
  - ロードショー準備

N期: 上場
  - 上場審査
  - ブックビルディング
  - 上場
```

### CFOが注意すべきIPOの論点

| 論点 | 内容 |
|------|------|
| 会計方針 | 収益認識、引当金計上基準の合理性 |
| 関連当事者取引 | 取引の合理性・必要性の説明 |
| 予実管理 | 予算と実績の乖離理由の説明能力 |
| 内部統制 | J-SOX対応（IT統制含む） |
| 資本政策 | ストック・オプション、株式分割 |
| 労務管理 | 未払残業代、36協定遵守 |
| 知的財産 | 競業避止、職務発明の帰属 |

## 6. M&A実務（CFO視点）

### バリュエーション

```
【DCF法】
企業価値 = Σ FCFt/(1+WACC)^t + TV/(1+WACC)^n
TV（ターミナルバリュー） = FCFn+1 / (WACC - g)

【マルチプル法】
EV/EBITDA法: EV = EBITDA × 業界マルチプル
EV/Revenue法: EV = Revenue × マルチプル（SaaS企業等）
PER法: 株式価値 = EPS × PER

【純資産法】
時価純資産 + 営業権（年買法: 営業利益 × 3-5年）
```

### PMI（Post Merger Integration）

```
Day 1-30: 安定化
  - ガバナンス体制の確立
  - 経理・財務機能の統合方針
  - 主要人材のリテンション

Day 31-100: 統合計画
  - 会計方針の統一
  - 勘定科目体系の統一
  - 管理会計（予算）の統合
  - 銀行口座の整理

Day 101-365: 本格統合
  - 基幹システムの統合検討
  - 内部統制の整備
  - シナジー効果の測定
```

## 7. 取締役会報告

### 月次報告パッケージ

```
【必須項目】
1. 月次業績サマリー（予実対比、前年対比）
2. キャッシュフロー報告（手元流動性推移）
3. 受注状況・パイプライン
4. KPIハイライト（稼働率、単価、NRR等）
5. リスク・課題報告
6. 今後のアクション

【四半期追加項目】
7. フォーキャスト更新（通期着地見込み）
8. セグメント別分析
9. 投資案件の進捗
10. 資本効率指標（ROIC、WACC、EVA）
```

### CEOへの財務アドバイス

- **数字で語る**: 定性的な議論を定量化する
- **選択肢を提示**: 「AかBか」ではなく「A/B/Cそれぞれのリスクとリターン」
- **悪いニュースは早く**: 業績下振れは早期にエスカレーション
- **長期視点**: 四半期業績だけでなく中長期の企業価値への影響
