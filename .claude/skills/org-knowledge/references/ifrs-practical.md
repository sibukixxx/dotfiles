# IFRS 実務ガイド（EY『International GAAP 2025』ベース）

## J-GAAP と IFRS の主要差異

### 差異の全体マップ

| 領域 | J-GAAP | IFRS | 影響 |
|------|--------|------|------|
| のれん | 20年以内で償却 | 非償却（毎期減損テスト） | 利益増、BS膨張 |
| 開発費 | 原則費用処理 | 要件充足で資産計上（IAS 38） | 利益増 |
| 有給休暇 | 引当計上不要が多い | IAS 19で未消化分を負債計上 | 負債増 |
| リース | ファイナンス/オペ区分 | 借手は原則全てオンバランス（IFRS 16） | 総資産・負債増 |
| 収益認識 | 基準29号（IFRS15に収斂済） | IFRS 15 | 差異縮小 |
| 金融商品分類 | 4分類 | IFRS 9: SPPI+ビジネスモデルテスト | 分類が変わりうる |
| 減損 | 2段階テスト（戻入なし） | 1段階（回収可能価額直接比較、戻入あり） | 損益変動 |
| 退職給付 | 遅延認識あり（回廊あり） | IAS 19R: 数理差異即時OCI | P/L平準化消失 |
| 非支配持分 | 取得後は純資産内 | IFRS 3: 全部のれん法も選択可 | のれん金額 |
| 表示 | 経常利益を重視 | 営業利益（IAS 1 → IFRS 18）が中心 | 段階利益の違い |

## IFRS 15 収益認識（深掘り）

### Rinが既に持つ5ステップモデルへの補足

#### 本人 vs 代理人（Principal vs Agent）

収益を総額か純額かで表示する判定:

```
【本人の指標（総額表示）】
✓ 財・サービスの支配を移転前に獲得する
✓ 在庫リスクを負う
✓ 価格設定の裁量がある

【代理人の指標（純額表示）】
✓ 手数料・手配のみ
✓ 在庫リスクを負わない
```

**SES企業の例**: プライムでの受注（本人）vs 再委託のコーディネーション（代理人判定が必要）

#### ライセンスの収益認識

| ライセンスの性質 | 認識タイミング | 例 |
|-----------------|---------------|-----|
| 使用権（Right to use） | 一時点 | パッケージソフト販売 |
| アクセス権（Right to access） | 一定期間 | SaaS、クラウドライセンス |

判定基準: 企業の活動がライセンスに重大な影響を及ぼすか

#### 契約変更（Contract Modification）

```
契約変更
├── 別個の契約として処理（追加財・サービスが別個 + 独立販売価格）
└── 既存契約の変更
    ├── 残存履行義務を将来に向かって修正
    └── 既存契約を解約＋新契約として処理
```

#### 返品権付き販売

- 収益: 返品が見込まれない部分のみ認識
- 返金負債: 返品見込み額を負債計上
- 返品資産: 返品される商品の回収見込額を資産計上

## IFRS 9 金融商品

### 分類と測定

```
金融資産の分類
├── 償却原価（AC）: SPPIテスト充足 + 回収目的のビジネスモデル
├── FVOCI（公正価値→OCI）: SPPIテスト充足 + 回収＋売却目的
├── FVPL（公正価値→P/L）: 上記以外、またはFVPL指定
└── FVOCI指定（株式）: 取消不能の選択（配当のみP/L、売却損益はOCIのまま）
```

**SPPIテスト**: 契約上のキャッシュフローが元本と利息のみで構成されているか

### 予想信用損失（ECL）モデル

| ステージ | トリガー | 引当金 |
|---------|---------|--------|
| Stage 1 | 当初認識時 | 12ヶ月ECL |
| Stage 2 | 信用リスクが著しく増大 | 全期間ECL |
| Stage 3 | 信用減損（デフォルト） | 全期間ECL（利息は純額） |

**簡便法**: 売上債権・契約資産 → 常に全期間ECLで測定可（引当金マトリクス）

### ヘッジ会計（IFRS 9）

| ヘッジ種類 | 対象 | 会計処理 |
|-----------|------|---------|
| 公正価値ヘッジ | 認識済資産/負債のFV変動リスク | ヘッジ対象・手段ともP/L |
| キャッシュフローヘッジ | 予定取引のCF変動リスク | 有効部分はOCI、非有効部分はP/L |
| 在外営業活動体ヘッジ | 純投資の為替リスク | 有効部分はOCI |

## IFRS 16 リース

### 借手の会計処理

```
リース開始日:
  使用権資産 XXX / リース負債 XXX

毎期:
  減価償却費 XXX / 使用権資産 XXX （定額法等）
  支払利息  XXX / リース負債 XXX （実効金利法）
```

- **割引率**: リースの計算利率（不明なら追加借入利率）
- **免除**: 短期リース（12ヶ月以下）、少額リース（新品時USD5,000以下目安）

### リース負債の再測定

- 指数・レートに基づく変動リース料の変更
- 残価保証の見積り変更
- リース期間の見直し（延長/解約オプションの評価変更）

## IAS 36 減損

### J-GAAPとの決定的な差異

| 項目 | J-GAAP | IFRS（IAS 36） |
|------|--------|----------------|
| 認識テスト | 割引前CF < 帳簿価額 | なし（直接比較） |
| 測定 | 帳簿価額 - 回収可能価額 | 同左 |
| 回収可能価額 | max(正味売却価額, 使用価値) | 同左 |
| のれんの減損 | 認識テストあり | CGU単位で毎期テスト |
| 戻入 | 不可 | のれん以外は可 |

### CGU（資金生成単位）

- 他の資産グループからおおむね独立したCFを生み出す最小の資産グループ
- のれんは恩恵を受けるCGU（グループ）に配分

## IAS 19 従業員給付

### 確定給付制度

```
正味確定給付負債（資産） = 確定給付債務(DBO) - 制度資産のFV

P/Lに計上:
  勤務費用（当期勤務費用 + 過去勤務費用 + 清算損益）
  正味利息（正味確定給付負債 × 割引率）

OCIに計上:
  数理計算上の差異（リサイクリングなし）
  制度資産の収益（正味利息除く）
```

**J-GAAPとの差異**: J-GAAPは数理差異の遅延認識が可能。IFRSは即時OCI認識（P/Lは平準化されない）

## IFRS 3 企業結合

### 取得法

```
のれん = 移転対価 + 非支配持分 + 既存持分のFV - 識別可能純資産のFV

非支配持分の測定（選択）:
  ├ 公正価値法（全部のれん法）→ 非支配持分にものれん配分
  └ 持分比例法（部分のれん法）→ 親会社持分のみ
```

### 条件付対価（アーンアウト）

- 取得日のFVで測定
- 事後の変動: 金融負債なら再測定差額はP/L

### 段階取得

- 既存持分を取得日のFVで再測定 → 差額はP/L
- のれんは「移転対価 + 既存持分FV + NCI」から算出

## IFRS 18 表示及び開示（2027年〜）

IAS 1を置き換える新基準:

### 損益計算書の新区分

```
営業区分（Operating）
  主たる事業活動の収益・費用
投資区分（Investing）
  投資活動からのリターン（利息・配当受取、投資先損益等）
財務区分（Financing）
  借入金利息、リース利息等
法人所得税
廃止事業
```

### 経営者業績指標（MPM）

- 経営者が使用する非GAAP指標（例: 調整後EBITDA）を注記で開示義務化
- IFRS上の最も直接比較可能な小計との調整表が必要

## IFRS適用時の実務チェックリスト

### 初度適用（IFRS 1）

- [ ] 開始B/Sの作成（移行日時点でIFRS認識・測定を遡及適用）
- [ ] 免除規定の選択判断（企業結合、累積為替差額、みなし原価等）
- [ ] 例外規定の確認（ヘッジ会計、非支配持分、金融資産分類等）
- [ ] 利益剰余金への累積的影響の調整

### J-GAAP → IFRS 調整仕訳の典型例

| 項目 | 調整内容 |
|------|---------|
| のれん償却 | 償却の戻入 → 利益増加 |
| 開発費 | 資産計上 → 利益増加（初期） |
| リース | オペリースのオンバランス化 → 資産・負債増 |
| 有給休暇 | 未消化分を負債計上 → 費用増加 |
| 退職給付 | 数理差異の即時認識 → 純資産変動 |
| 減損戻入 | 過去の減損を一部戻入可能 → 利益増加 |
