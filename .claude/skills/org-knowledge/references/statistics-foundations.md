# 統計学の基礎と応用（参考文献体系）

## 記述統計と確率分布（『統計学入門』東大教養ベース）

### データの要約

| 統計量 | 定義 | 用途 |
|--------|------|------|
| 平均値 (x̄) | Σxi / n | 中心的傾向（外れ値に敏感） |
| 中央値 | 順位 (n+1)/2 の値 | 中心的傾向（ロバスト） |
| 分散 (s²) | Σ(xi - x̄)² / (n-1) | ばらつき（不偏推定量） |
| 標準偏差 (s) | √s² | ばらつき（元の単位） |
| 四分位範囲 (IQR) | Q3 - Q1 | ロバストなばらつき |
| 歪度 | E[(X-μ)³] / σ³ | 分布の非対称性 |
| 尖度 | E[(X-μ)⁴] / σ⁴ - 3 | 裾の重さ |

### 主要な確率分布

#### 離散分布

| 分布 | パラメータ | 平均 | 分散 | 用途 |
|------|-----------|------|------|------|
| ベルヌーイ | p | p | p(1-p) | 成功/失敗 |
| 二項分布 | n, p | np | np(1-p) | n回試行の成功数 |
| ポアソン分布 | λ | λ | λ | 稀な事象の発生数 |
| 幾何分布 | p | 1/p | (1-p)/p² | 初回成功までの試行数 |
| 負の二項分布 | r, p | r/p | r(1-p)/p² | r回成功までの失敗数 |

#### 連続分布

| 分布 | パラメータ | 特徴 | 用途 |
|------|-----------|------|------|
| 正規分布 | μ, σ² | 釣鐘型、中心極限定理 | 自然現象、測定誤差 |
| t分布 | ν（自由度） | 正規より裾が重い | 小標本の平均検定 |
| χ²分布 | ν | 非負、右に歪む | 分散の検定、適合度 |
| F分布 | ν₁, ν₂ | 2つのχ²の比 | 分散比の検定、ANOVA |
| 指数分布 | λ | 無記憶性 | 事象間の待ち時間 |
| ガンマ分布 | α, β | 指数の一般化 | 待ち時間の合計 |
| ベータ分布 | α, β | [0,1]上の分布 | 確率の確率（ベイズ事前分布） |
| 対数正規分布 | μ, σ² | log(X)が正規 | 所得分布、株価 |

### 推定

```
【点推定】
最尤推定量 (MLE): L(θ|x) = Π f(xi|θ) を最大化
モーメント法: 母集団モーメント = 標本モーメント

【区間推定】
信頼区間: θ̂ ± z_{α/2} × SE(θ̂)
  正規母集団、σ既知: x̄ ± z_{α/2} × σ/√n
  正規母集団、σ未知: x̄ ± t_{α/2,n-1} × s/√n
  大標本（CLT適用）: x̄ ± z_{α/2} × s/√n
  比率: p̂ ± z_{α/2} × √(p̂(1-p̂)/n)
```

### 仮説検定

```
【手順】
1. 帰無仮説 H₀ と対立仮説 H₁ を設定
2. 有意水準 α を決定（通常 0.05 or 0.01）
3. 検定統計量を計算
4. p値を算出（または棄却域と比較）
5. 判定: p < α → H₀を棄却

【第一種・第二種の過誤】
  第一種の過誤（α）: H₀が真なのに棄却（偽陽性）
  第二種の過誤（β）: H₀が偽なのに棄却しない（偽陰性）
  検出力 = 1 - β

【主要な検定】
  1標本t検定: 母平均の検定
  2標本t検定: 2群の平均差（等分散/Welch）
  対応のあるt検定: 対応データの平均差
  χ²検定: 適合度検定、独立性検定
  F検定: 等分散性の検定
  Wilcoxon検定: ノンパラメトリック（順位ベース）
  Mann-Whitney U検定: 2群のノンパラメトリック比較
```

### 多重比較

- **ボンフェローニ補正**: α/m（保守的）
- **Holm法**: ステップダウン法（ボンフェローニより検出力高い）
- **BH法（Benjamini-Hochberg）**: FDR制御（大規模検定向け）

## 回帰分析（佐和隆光『回帰分析』ベース）

### 線形回帰モデル

```
y = Xβ + ε
  y: 応答変数（n×1）
  X: 計画行列（n×p）
  β: 回帰係数（p×1）
  ε: 誤差項（n×1）、ε ~ N(0, σ²I)

OLS推定量: β̂ = (X'X)⁻¹X'y
  E[β̂] = β（不偏性）
  Var[β̂] = σ²(X'X)⁻¹
  ガウス・マルコフの定理: OLSはBLUE（最良線形不偏推定量）
```

### 回帰診断

| 診断 | 方法 | 対処 |
|------|------|------|
| 線形性 | 残差プロット、偏回帰プロット | 変数変換、多項式項追加 |
| 等分散性 | 残差 vs 予測値プロット、BP検定 | WLS、ロバスト標準誤差 |
| 正規性 | Q-Qプロット、Shapiro-Wilk検定 | 変数変換、一般化線形モデル |
| 独立性 | Durbin-Watson統計量 | 時系列モデル、GLS |
| 多重共線性 | VIF > 10 | 変数選択、正則化、PCA |
| 外れ値 | Cook's距離、レバレッジ | ロバスト回帰、外れ値除去 |

### 変数選択

```
【情報量規準】
AIC = -2 ln L + 2k        # 予測重視
BIC = -2 ln L + k ln(n)   # モデル選択（BICはより簡素なモデルを好む）
調整済みR² = 1 - (1-R²)(n-1)/(n-k-1)

【逐次選択法】
前進選択法: 変数を1つずつ追加
後退消去法: 全変数から1つずつ除去
ステップワイズ法: 追加と除去を繰り返す
→ 現代は正則化（Lasso/Ridge）を推奨
```

### 一般化線形モデル（GLM）

```
g(E[Y]) = Xβ
  g: リンク関数
  Y: 指数型分布族

【主要なGLM】
| 応答変数の分布 | リンク関数 | 用途 |
|---------------|-----------|------|
| 正規分布 | 恒等 (identity) | 通常の線形回帰 |
| 二項分布 | ロジット (logit) | ロジスティック回帰 |
| ポアソン分布 | 対数 (log) | カウントデータ |
| ガンマ分布 | 逆数 (inverse) | 正値連続データ |
| 逆ガウス分布 | 1/μ² | 応答時間データ |
```

## 統計的学習理論（ISLR / ESL ベース）

### バイアス-バリアンス分解

```
E[(y - f̂(x))²] = Var(f̂(x)) + [Bias(f̂(x))]² + Var(ε)
                   ↑バリアンス    ↑バイアス²     ↑削減不能

モデル複雑度↑ → バイアス↓、バリアンス↑
モデル複雑度↓ → バイアス↑、バリアンス↓
→ トレードオフの最適点を見つける（交差検証で推定）
```

### 正則化

```
【Ridge回帰（L2正則化）】
β̂_ridge = argmin { ||y - Xβ||² + λ||β||² }
→ 係数を縮小するが0にはしない
→ 多重共線性に有効

【Lasso回帰（L1正則化）】
β̂_lasso = argmin { ||y - Xβ||² + λ||β||₁ }
→ 係数をスパースに（変数選択効果）
→ 高次元データに有効

【Elastic Net（L1+L2）】
β̂_enet = argmin { ||y - Xβ||² + λ₁||β||₁ + λ₂||β||² }
→ 相関する変数群を同時に選択
```

### 交差検証

```
【k-fold CV】
1. データをk個に分割
2. k-1個で学習、1個で評価をk回繰り返す
3. k個のテスト誤差の平均を推定誤差とする

k=5 or k=10 が標準
k=n: Leave-One-Out CV（LOO-CV）→ ほぼ不偏だがバリアンス大

【時系列データ】
通常のCVは不適切（時間的な依存構造を破壊）
→ 時系列CV: ローリングウィンドウ or 拡張ウィンドウ
```

### 主要な教師あり学習手法

| 手法 | 特徴 | 長所 | 短所 |
|------|------|------|------|
| 線形回帰 | y = Xβ + ε | 解釈性、推論可能 | 非線形に弱い |
| ロジスティック回帰 | P(Y=1) = logistic(Xβ) | 確率推定、解釈性 | 線形決定境界 |
| k近傍法 (KNN) | 近傍k個の多数決/平均 | ノンパラメトリック | 次元の呪い |

| ランダムフォレスト | 決定木のバギング+特徴量ランダム選択 | 高精度、ロバスト | 解釈性低下 |
| 勾配ブースティング | 残差に逐次的にフィット | 最高精度クラス | チューニング必要 |

### 次元削減

```
【PCA（主成分分析）】
共分散行列の固有値分解
  S = VΛV'

【因子分析】

【t-SNE / UMAP】

```

## ベイズ統計（安道知寛『ベイズ統計モデリング』/ 松浦『RとStanではじめる ベイズ統計モデリング』ベース）

### ベイズの定理

```
事後分布 ∝ 尤度 × 事前分布
p(θ|y) ∝ p(y|θ) × p(θ)

【ベイズ推論の流れ】
1. モデルを設定: y ~ p(y|θ)
2. 事前分布を設定: θ ~ p(θ)
3. 事後分布を計算/近似: p(θ|y)
4. 事後分布から推論: 点推定、区間推定（信用区間）、予測分布
```

### 共役事前分布

| 尤度 | 共役事前分布 | 事後分布 |
|------|------------|---------|
| 二項分布 | Beta(α, β) | Beta(α+y, β+n-y) |
| ポアソン分布 | Gamma(α, β) | Gamma(α+Σy, β+n) |
| 正規分布（μ, σ²既知） | Normal(μ₀, σ₀²) | Normal(μ_n, σ_n²) |
| 正規分布（σ²） | Inverse-Gamma(α, β) | Inverse-Gamma(α+n/2, β+SS/2) |
| 多項分布 | Dirichlet(α) | Dirichlet(α+y) |

### MCMC（マルコフ連鎖モンテカルロ法）

```
【目的】
事後分布 p(θ|y) からの直接サンプリングが困難な場合に
マルコフ連鎖を構成して事後分布からのサンプルを近似的に得る

【主要アルゴリズム】
メトロポリス・ヘイスティングス法:
  1. 提案分布 q(θ*|θ_t) からθ*を生成
  2. 採択確率 α = min(1, p(θ*|y)q(θ_t|θ*) / p(θ_t|y)q(θ*|θ_t))
  3. 確率αで採択、1-αで棄却

ギブスサンプリング:
  各パラメータを条件付き分布から順次サンプリング
  θ₁ ~ p(θ₁|θ₂,...,θ_k, y)
  θ₂ ~ p(θ₂|θ₁,θ₃,...,θ_k, y)
  ...

ハミルトニアンモンテカルロ法 (HMC):
  勾配情報を使った効率的なサンプリング
  → Stan の NUTS（No-U-Turn Sampler）が実装

【収束診断】
R̂（Gelman-Rubin統計量）: < 1.01 が目標
実効サンプルサイズ (ESS): 十分大きいか確認
トレースプロット: 定常状態に達しているか目視
```

### Stan によるモデリング

```stan
data {
  int<lower=0> N;
  vector[N] x;
  vector[N] y;
}
parameters {
  real alpha;
  real beta;
  real<lower=0> sigma;
}
model {
  // 事前分布
  alpha ~ normal(0, 10);
  beta ~ normal(0, 10);
  sigma ~ cauchy(0, 5);

  // 尤度
  y ~ normal(alpha + beta * x, sigma);
}
generated quantities {
  vector[N] y_rep;
  for (n in 1:N)
    y_rep[n] = normal_rng(alpha + beta * x[n], sigma);
}
```

### 階層ベイズモデル

```
【構造】
データレベル:   y_ij ~ p(y|θ_j)
グループレベル: θ_j ~ p(θ|φ)
ハイパーパラメータ: φ ~ p(φ)

【利点】
- 情報の借用（shrinkage）: データの少ないグループが全体の情報を借りる
- 部分的プーリング: 完全プーリングとノープーリングの中間
- 過適合の抑制

【例: 学校別テストスコア（Eight Schools）】
y_j ~ Normal(θ_j, σ_j)
θ_j ~ Normal(μ, τ)
μ ~ Normal(0, 100)
τ ~ HalfCauchy(0, 25)
```

## 統計的因果推論（Pearl『入門 統計的因果推論』/ ローゼンバウム『統計的因果推論入門』ベース）

### 因果推論の枠組み

```
【潜在結果（Rubin の枠組み）】
個体iの処置効果: τ_i = Y_i(1) - Y_i(0)
  Y_i(1): 処置を受けた場合の結果
  Y_i(0): 処置を受けなかった場合の結果
→ 因果推論の根本問題: 同一個体で両方を観測できない

【平均処置効果】
ATE = E[Y(1) - Y(0)]
ATT = E[Y(1) - Y(0) | T=1]  # 処置群における効果
ATU = E[Y(1) - Y(0) | T=0]  # 非処置群における効果
```

### DAG（有向非巡回グラフ）と因果構造

```
【基本構造】
連鎖（チェーン）: X → Z → Y
  Zで条件付けると X ⊥ Y | Z

分岐（フォーク）: X ← Z → Y
  Zで条件付けると X ⊥ Y | Z（交絡の除去）

合流（コライダー）: X → Z ← Y
  条件付けしない: X ⊥ Y
  Zで条件付けると: X ⫫̸ Y | Z（コライダーバイアス）

【バックドア基準】
Zの集合がXからYへのバックドアパスを全てブロックし、
かつXの子孫を含まない → Zで条件付ければ因果効果を識別可能

【フロントドア基準】
X → M → Y で、X→Yの直接パスがなく、M経由のみ
→ Mを介して因果効果を識別
```

### 傾向スコア

```
e(x) = P(T=1|X=x)  # 処置を受ける確率

【利用法】
1. マッチング: e(x)が近い処置群・対照群をペアリング
2. IPW（逆確率重み付け）: 重み w = T/e(x) + (1-T)/(1-e(x))
3. 層別化: e(x)で層を作り層内で効果推定
4. 二重にロバストな推定量: IPW + 結果モデルの組合せ

【チェック】
バランスチェック: マッチング/重み付け後の共変量バランス
  標準化平均差 (SMD) < 0.1 が目安
```

### 主要な因果推論手法

| 手法 | 想定 | 識別条件 |
|------|------|---------|
| ランダム化実験（RCT） | 処置のランダム割当 | 金本位（最強） |
| 回帰調整 | 交絡変数を全て観測 | 条件付き独立性 |
| 傾向スコアマッチング | 〃 | 〃 + 共通サポート |
| IPW（逆確率重み付け） | 〃 | 〃 + 正の確率 |
| 操作変数法（IV） | 未観測交絡あり | 除外制約、関連性 |
| 差分の差分法（DID） | パネル/反復横断 | 平行トレンド仮定 |
| 回帰不連続デザイン（RDD） | 閾値による割当 | 閾値周辺での連続性 |
| 合成統制法 | 集計的介入評価 | 処置前フィットの良さ |

### do演算と介入

```
Pearl の do 演算:
  P(Y|do(X=x)) ≠ P(Y|X=x)

左辺: Xをxに介入したときのYの分布（因果効果）
右辺: X=xが観測されたときのYの条件付き分布（相関）

バックドア調整:
  P(Y|do(X=x)) = Σ_z P(Y|X=x, Z=z) P(Z=z)
```

## 予測モデリング（『予測にいかす統計モデリングの基本』ベース）

### 予測の評価指標

| 指標 | 定義 | 特徴 |
|------|------|------|
| RMSE | √(Σ(y-ŷ)²/n) | 二乗誤差、外れ値に敏感 |
| MAE | Σ|y-ŷ|/n | 絶対誤差、ロバスト |
| MAPE | Σ|y-ŷ|/y × 100/n | 相対誤差（0近辺で不安定） |
| R² | 1 - SS_res/SS_tot | 決定係数（説明力） |
| AUC | ROC曲線下面積 | 分類モデルの総合指標 |
| 対数損失 | -Σ(y log ŷ + (1-y) log(1-ŷ))/n | 確率予測の評価 |

### 過適合の防止

```
1. ホールドアウト（訓練/検証/テスト分割）
2. 交差検証（k-fold CV）
3. 正則化（Ridge, Lasso, Elastic Net）
4. 早期停止（ブースティング、ニューラルネット）
5. モデルの簡素化（特徴量選択、次元削減）
6. アンサンブル（バギング、ブースティング、スタッキング）
```

## 時系列分析（『状態空間時系列分析入門』ベース）

### 古典的時系列モデル

```
【ARIMA(p,d,q)モデル】
AR(p): y_t = c + φ₁y_{t-1} + ... + φ_py_{t-p} + ε_t
MA(q): y_t = c + ε_t + θ₁ε_{t-1} + ... + θ_qε_{t-q}
ARMA(p,q): AR + MA の組合せ
ARIMA(p,d,q): d回差分をとってARMA

【モデル選定（Box-Jenkins法）】
1. 定常性の確認（ADF検定、KPSS検定）
2. 差分の次数 d を決定
3. ACF/PACFから p, q を推定
4. AIC/BICで最適モデル選択
5. 残差診断（Ljung-Box検定）

【季節性ARIMA: SARIMA(p,d,q)(P,D,Q)_s】
```

### 状態空間モデル

```
【一般形】
観測方程式: y_t = Z_t α_t + ε_t,  ε_t ~ N(0, H_t)
状態方程式: α_{t+1} = T_t α_t + R_t η_t,  η_t ~ N(0, Q_t)

  y_t: 観測値
  α_t: 状態ベクトル（トレンド、季節成分等の潜在変数）
  Z_t: 観測行列
  T_t: 遷移行列
  H_t: 観測ノイズの分散
  Q_t: システムノイズの分散

【カルマンフィルタ】
予測ステップ:
  α_{t|t-1} = T_t α_{t-1|t-1}
  P_{t|t-1} = T_t P_{t-1|t-1} T_t' + R_t Q_t R_t'

更新ステップ:
  K_t = P_{t|t-1} Z_t' (Z_t P_{t|t-1} Z_t' + H_t)⁻¹
  α_{t|t} = α_{t|t-1} + K_t (y_t - Z_t α_{t|t-1})
  P_{t|t} = (I - K_t Z_t) P_{t|t-1}

【カルマンスムーザ】
t=T から逆方向に平滑化 → 全データを使った最適推定
```

### 状態空間モデルの主要モデル

| モデル | 構造 | 用途 |
|--------|------|------|
| ローカルレベルモデル | トレンド（ランダムウォーク） | 水準の変動 |
| ローカル線形トレンド | レベル + 傾き | トレンド変化の追跡 |
| 季節モデル | 季節成分を状態に含む | 季節調整 |
| 構造時系列モデル | トレンド + 季節 + 回帰 | 複合的な時系列分解 |
| 動的回帰モデル | 回帰係数が時変 | 関係性の変化を追跡 |

### ベイズ的時系列（Stan + 状態空間モデル）

```
【利点】
- パラメータの不確実性を自然に扱える
- 事前分布で構造的知識を導入
- 欠測値の自然な扱い
- モデル比較（WAIC, LOO-CV）

【実装例（ローカルレベルモデル）】
data {
  int<lower=0> T;
  vector[T] y;
}
parameters {
  vector[T] mu;        // 状態
  real<lower=0> sigma_obs;  // 観測ノイズ
  real<lower=0> sigma_sys;  // システムノイズ
}
model {
  sigma_obs ~ cauchy(0, 5);
  sigma_sys ~ cauchy(0, 5);
  mu[1] ~ normal(y[1], 10);
  for (t in 2:T)
    mu[t] ~ normal(mu[t-1], sigma_sys);
  y ~ normal(mu, sigma_obs);
}
```

### 変化点検出

- **構造変化検定**: Chow検定、CUSUM検定
- **ベイズ変化点**: 変化点の事後分布を推定
- **Prophet**: Facebookの時系列予測（トレンド変化点を自動検出）
